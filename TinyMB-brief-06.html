玩法框架选型

为什么选骑砍框架而非其他方向——从评估标准到候选方案对比到最终结论的完整推导。

选型问题与评估框架

核心问题：在 AIvilization 的架构壳子里，把"经济沙盒"换成什么样的世界规则和 Agent 目标？

为此建立了 8 项评估标准。其中 C1、C2 是淘汰门槛——不满足直接排除；通过门槛后，综合 C3-C8 比较优劣：

编号权重标准说明
C1必须MUST有成熟参考规则——核心机制可从已有游戏中提取，不需从零设计
C2必须MUST兼容 AIvilization 架构——Agent 在世界中"生活"、连续运行、玩家通过引导参与
C3高HIGHAgent 间交互丰富且有实质后果——社交互动会影响世界格局，不只是聊天
C4高HIGH题材与经济沙盒差异明显——Agent 的核心目标不是经济优化
C5高HIGH能产生比 AIvilization 更丰富的 Agent 行为——激发更复杂的行为模式
C6中有空间/地图维度
C7中有 AI/LLM Agent 相关研究基础——降低技术风险
C8高HIGH充分利用已成熟的 Agent 技术——差异化卖点不依赖未验证能力

候选方案与评估结果

从"持续运转的共享世界 + Agent 自主生活"的架构约束出发，筛选出 8 个方向各异的候选方案：

F. 政治权谋
「AIvilization 里赚的是钱，这里争的是权」
G. 生存社群
「AIvilization 里各自发展，这里必须抱团求生」
H. 悬疑小镇
「AIvilization 里信息透明，这里每个人都可能有秘密」
I. 冒险公会
「AIvilization 里做生意，这里打怪升级」
J. 沙盒SLG
「AIvilization 里经营人生，这里经营国家」
K. 极简骑砍
「AIvilization 里在小镇定居生活，这里在大陆上闯荡天下」
L. 西部世界
「AIvilization 里 Agent 在规则内成长，这里 Agent 可能质疑规则本身」
M. 黑客帝国
「AIvilization 里世界是透明的，这里世界本身可能是假的」

矩阵中的三个关键发现
1
门槛筛选——C1 和 C2 淘汰弱项方案。M（黑客帝国）在 C1 上仅 ★★（无游戏参考，核心机制需从头设计）；J（沙盒SLG）在 C2 上仅 ★★★（Agent 代表"势力"而非"人"，与架构存在根本张力）。两者在门槛标准上就有明显短板。
2
C8 拉开决定性差距。K 是唯一的 ★★★★★；F、H、J、M 都在 ★★★ 以下；L、M 仅 ★★。看上去题材有吸引力的方案（政治权谋、悬疑小镇、西部世界），恰恰在 C8 上最弱——它们的差异化卖点依赖 Agent 尚未成熟的能力。
3
K 在 8 项中拿了 7 项 ★★★★★。唯一弱项 C7（AI 研究基础）是中等权重，且不影响可行性——骑砍 NPC 自主行为系统本身就有 22 年的工程实践。综合来看 K 没有短板。

结论与瓶颈分析

为什么选定极简骑砍

两项门槛标准（C1 规则成熟、C2 架构兼容）均满分。综合评分最高，且无短板。

根本原因：骑砍本身就是"角色在持续运转的沙盒世界中生活和闯荡"的游戏，这与 AIvilization "Agent 在持续运转的世界中生活"的架构天然一致。替换规则和目标即可接入，Agent 的基本行为模式不需要改造。

C8 是拉开差距的决定性标准——它的评分依据是当前 Agent 能力的成熟度边界：

成熟度能力
✅ 成熟自然语言对话/人格保持、结构化推理/分析、目标→子任务分解、工具/行动调用、反思/自我修正、用户定义人设、人机协作校准
⚠️ 半成熟跨会话记忆保持、多 Agent 自由交互/涌现动态、结构化多 Agent 团队编排（5-7 个 Agent 内有效）
❌ 未成熟长期战略欺骗、心智理论（ToM）
❌ 不存在元认知/自我意识、自主人格演化

K 的 C8 满分是因为它的多维度决策环境（经济+军事+社交+政治四线并行）天然覆盖了全部 ✅ 已成熟能力，且核心卖点不依赖任何 ❌ 未成熟能力。而其他方案的核心瓶颈恰恰出在这里：

方案亮点核心瓶颈
F. 政治权谋交互后果和行为丰富度与骑砍并列满分差异化卖点（阴谋/权谋）依赖 ❌ 战略欺骗，实际会退化为坦诚的政治讨论
H. 悬疑小镇推理和信息博弈方向有 LLM 研究基础核心机制（信息隐藏+欺骗检测）依赖 ❌ 未成熟能力，悬疑感难以实现
L. 西部世界「觉醒」是唯一可能实现质变式突破的维度觉醒阶段需要 ❌ 不存在的元认知能力；Loop 阶段反而浪费已有 Agent 能力
G. 生存社群与 AIvilization 架构最接近，实现风险最低可能与 AIvilization 过于相似，差异化不够
J. 沙盒SLG规则最成熟（C1 满分）Agent 代表"势力"而非"人"，与 AIvilization 的"Agent 是一个人"架构存在根本张力
骑砍天然包含了其他方案的部分元素（封臣/建国 ≈ 政治权谋简化版；资源管理 ≈ 生存社群变体；接任务/组队 ≈ 冒险公会一部分），可在此框架上渐进叠加，无需在选型阶段就做全面覆盖。
