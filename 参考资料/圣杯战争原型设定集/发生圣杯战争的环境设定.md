# 发生圣杯战争的环境设定

> 本文档综合 Fate 系列各作品，提炼圣杯战争发生所需的世界观基底、地理环境、组织势力和社会背景。重点是抽象出"什么样的世界/环境会催生圣杯战争"这一设定框架。

---

## 一、世界观基底

### 1.1 神秘（Mystery）

Fate 世界的根基法则——"神秘"——指未被大众认知的超自然力量体系。神秘的核心特性：

- **越不为人知，力量越强**：广泛认知的"神秘"将失去力量，因此魔术世界将"防止神秘泄露"视为最高准则。
- **越古老越强大**：远古神代的神秘远超现代，英灵的力量来源于其传说中积累的信仰与神秘。
- **现代魔术的衰退**：随着科学扩张和"神秘"消退，现代魔术师的能力远不及古代；圣杯战争正是在这种背景下诞生的——魔术师为到达根源而不得不借助仪式的力量。

### 1.2 根源（Root / Akasha）

- 存在于世界外侧的"万物之开始与终结"。
- 记录世上一切、创造世上一切的"神之座"。
- 所有魔术师的终极目标——到达根源，意味着触碰世界的本质真理。
- 圣杯战争的深层目的就是利用从者灵魂打开通往根源的通道。

### 1.3 英灵之座（Throne of Heroes）

- 位于世界外侧、脱离时间轴的超越性空间。
- 英雄死后，其灵魂因丰功伟绩和人们的信仰而升格为"英灵"，被收纳于英灵之座。
- 英灵之座中的英灵可以被召唤至任何时代——过去或未来。
- 从者是英灵本体的"分身/复制"，战败消亡后灵魂回归英灵之座，英灵本体可以像阅读书籍一样获知分身的经历。

## 二、地理环境要素

### 2.1 灵脉（Ley Line）

灵脉是圣杯战争发生的**首要地理条件**。

| 要素 | 说明 |
|------|------|
| 本质 | 地球表面流动的魔力能量网络 |
| 作用 | 大圣杯从灵脉中汲取和储存魔力，是仪式的能量来源 |
| 强度要求 | 圣杯战争需要"世所罕见的强大灵脉"作为支撑，普通灵脉无法提供足够的魔力 |
| 抽取节奏 | 为避免灵脉枯竭，魔力抽取极其缓慢，约需 60 年蓄满一次 |

冬木市之所以成为经典舞台，正是因为其拥有异常强大的灵脉。

### 2.2 灵脉汇聚点与关键地标

圣杯战争的舞台通常包含以下类型的关键地点：

| 地标类型 | 功能 | 冬木市对应 |
|----------|------|-----------|
| 大圣杯所在地 | 仪式核心，大规模魔法阵的安置处；通常位于灵脉最强的地下空洞 | 柳洞寺（圆藏山）地下"龙洞" |
| 灵脉强点 | 从者交战或建立据点的理想地点，因魔力充裕可增强从者战力 | 柳洞寺、远坂宅邸、新都中心 |
| 监督者驻地 | 圣堂教会的教堂，作为中立地带提供战败者庇护 | 冬木教会 |
| 城市区域 | 战场范围，需要维持"神秘隐匿" | 冬木市（深山町、新都等） |

### 2.3 城市特征

承办圣杯战争的城市通常具有以下特征：

- **魔术基础设施**：由创始家族长期经营的魔术结界和管理体系。
- **适度规模**：城市足够大以容纳多组 Master-Servant 的行动空间，但不至于大到无法隐匿战斗痕迹。
- **隐秘性**：非魔术界的主流城市，不容易引起外界关注。
- **双重面貌**：白天是正常的人类都市，夜晚成为从者交战的暗战场。

## 三、组织势力

### 3.1 创始御三家

圣杯战争的"设计者"，共同策划并建立了整套仪式系统。三家各自负责不同的核心功能：

| 家族 | 负责领域 | 核心贡献 |
|------|----------|---------|
| 爱因兹贝伦 | 天之杯（第三魔法） | 提供大圣杯的中枢"冬之圣女"、制作小圣杯载体（人造人） |
| 远坂 | 灵脉管理 | 提供冬木市的灵脉作为仪式场地，维护灵脉结界 |
| 玛奇里（间桐） | 令咒系统 | 设计令咒的结构和从者控制机制 |

御三家在每次圣杯战争中拥有天然的参战优先权和情报优势。

### 3.2 魔术协会

| 属性 | 描述 |
|------|------|
| 本质 | 由不同国籍、流派的魔术师组成的自卫团体 |
| 使命 | 管理、隐匿和发展魔术 |
| 最高准则 | 防止"神秘的泄露" |
| 总部 | 时钟塔（英国伦敦） |
| 对圣杯的态度 | 将圣杯视为到达"根源"的手段，积极派遣或资助魔术师参战 |

**魔术协会三大部门**：

| 部门 | 位置 | 特征 |
|------|------|------|
| 时钟塔 | 英国伦敦 | 协会本部，由 12 位贵族和 12 个学部管理，西欧九成魔术师聚集于此 |
| 阿特拉斯院 | 埃及阿特拉斯山脉 | 炼金术师聚集地，奉行严格秘密主义 |
| 彷徨海 | 北欧 | 海上移动的山脉形组织，最古老的分部 |

### 3.3 圣堂教会

| 属性 | 描述 |
|------|------|
| 本质 | 以信仰为基础的宗教武装组织 |
| 与魔术协会的关系 | 表面互不侵犯，实际暗中持续对立 |
| 对圣杯的态度 | 名义上追求真正的圣遗物，实际上派遣监督者介入圣杯战争以监控和收集情报 |
| 核心角色 | 派遣"监督者"确保圣杯战争的秩序 |

### 3.4 监督者（Overseer / Supervisor）

| 职能 | 描述 |
|------|------|
| 规则维护 | 确保战争按基本规则进行 |
| 神秘隐匿 | 协调善后工作，抹消战斗痕迹和普通人记忆 |
| 败退保护 | 为弃权或落败的 Master 提供教会庇护 |
| 情报掌控 | 通过"灵器盘"掌握已召唤从者的数量和职阶 |
| 中立义务 | 名义上中立，但历史上多次出现监督者暗中参战的情况 |

## 四、社会规则与隐匿体制

### 4.1 "里侧"与"表侧"的双层社会

Fate 世界中，普通人生活在"表侧"——不知晓魔术存在的日常世界。魔术师和教会等势力在"里侧"活动。圣杯战争发生在两层之间的夹缝中：

- 白天：城市正常运转，Master 伪装为普通人。
- 夜晚：从者现身战斗，战场多选在人迹罕至的地方（公园、港口、郊外等）。
- 善后：监督者与魔术师协作，对目击者进行记忆篡改，修复物理破坏痕迹，向媒体提供虚假解释（煤气爆炸、自然灾害等）。

### 4.2 神秘隐匿的执行

| 执行手段 | 描述 |
|----------|------|
| 结界 | 在战场周围展开结界，排斥普通人进入或使其无法感知异常 |
| 记忆操作 | 对目击者施加暗示魔术或催眠，篡改记忆 |
| 物理伪装 | 将超自然破坏伪装为自然灾害、事故等 |
| 时间选择 | 将大规模战斗安排在深夜或人迹罕至的地点 |
| 灵体化 | 从者在非战斗时切换为灵体形态，对普通人完全不可见 |

## 五、魔术师社会的基本设定

### 5.1 魔术回路（Magic Circuit）

| 属性 | 描述 |
|------|------|
| 本质 | 魔术师体内的模拟神经，将生命力转换为魔力的器官 |
| 数量 | 生来决定，无法通过外在手段增加 |
| 功能 | 从外界汲取魔力并转换为人类可使用的形态；连接大魔术式 |
| 意义 | 拥有魔术回路是成为魔术师（以及 Master）的必要资质 |

### 5.2 魔术师的价值观

| 价值维度 | 描述 |
|----------|------|
| 终极追求 | 到达"根源"——凌驾于一切之上的目标 |
| 代际传承 | 魔术刻印代代相传，家族历史越长积累越深 |
| 神秘至上 | 将隐匿神秘置于道德和人命之上 |
| 实用主义 | 为目标可以牺牲一切，包括自身和他人的生命 |
| 竞争意识 | 家族间争夺稀有资源（灵脉、魔术基盘、圣遗物等） |

## 六、时间与周期

### 6.1 圣杯战争的时间节奏

| 阶段 | 时长 | 描述 |
|------|------|------|
| 魔力积蓄期 | ~60 年 | 大圣杯从灵脉缓慢汲取魔力 |
| Master 选定期 | 数月至一年 | 大圣杯标记候选人，候选人准备触媒和仪式 |
| 召唤期 | 数日至数周 | 七骑从者陆续被召唤 |
| 交战期 | 通常 2-4 周 | 从七骑齐聚到最终决战 |

### 6.2 战争的节律特征

- **夜间活跃**：从者战斗主要发生在夜晚，白天用于休整、情报收集和战略策划。
- **前期试探**：战争初期各方互相试探，收集情报，谨慎行动。
- **中期结盟/对抗**：随着情报积累，出现临时同盟和集中火力的目标。
- **后期决战**：存活组数减少后，大规模决战频繁发生，战斗烈度急剧上升。

## 七、不同作品中的环境变体

### 7.1 冬木市（Fate/stay night、Fate/Zero）

经典舞台。日本地方都市，拥有强大灵脉。柳洞寺下方安置大圣杯。历经五次圣杯战争。第三次战争后大圣杯被"此世全部之恶"（安哥拉曼纽）污染，导致后续战争中圣杯的愿望实现被附加恶意。

### 7.2 图里法斯（Fate/Apocrypha）

罗马尼亚小镇。大圣杯被尤格德米雷尼亚家族从冬木运至此地。因环境改变，圣杯战争变异为两大阵营的集团战。

### 7.3 Moon Cell（Fate/Extra）

月球上的光子结晶超级计算机内部。完全虚拟的数字空间。从者以数据形态存在。采用淘汰赛制而非自由混战。

### 7.4 特异点（Fate/Grand Order）

不是传统意义的"圣杯战争舞台"，而是历史被扭曲的异常时空节点。圣杯作为历史扭曲的催化剂出现在各特异点中。迦勒底的 Master 通过灵子转移前往修复。

## 八、环境设定的核心抽象

无论具体作品如何变化，圣杯战争的环境设定始终包含以下核心要素：

| 环境要素 | 抽象描述 |
|----------|---------|
| 能量源 | 需要一个庞大的魔力/能量供给系统（灵脉、Moon Cell 的计算力等） |
| 封闭战场 | 需要一个相对封闭、可控的区域作为战场 |
| 隐匿机制 | 需要一套将战争隔绝于普通世界的遮蔽体系 |
| 仲裁者 | 需要一个（名义上）中立的裁判角色维持基本秩序 |
| 势力博弈 | 需要多方组织（设计者、监管者、外部势力）的利益纠葛 |
| 双层社会 | 普通人的日常世界与战争暗面共存 |
| 周期性 | 战争不是一次性事件，而是周期性重复的仪式 |
