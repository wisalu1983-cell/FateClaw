# Master 与 Servant

> 本文档综合 Fate 系列各作品，提炼 Master（御主）和 Servant（从者）的核心设定、能力体系、关系机制，以及双方在圣杯战争中典型的行动模式。

---

## 第一部分：Servant（从者）

### 一、从者的本质

| 属性 | 描述 |
|------|------|
| 定义 | 通过仪式从"英灵之座"召唤至现世的英灵分身 |
| 本体关系 | 从者是英灵本体的"复制品"，使用英灵的信息制作而成。并非英灵本体。 |
| 存在形态 | 从者获得由魔力构成的肉体，可在灵体和实体之间切换 |
| 记忆回归 | 从者消亡后，构成其的信息（灵魂）回归英灵本体。英灵可如阅读书籍般获知分身的经历 |
| 召唤条件 | 需要"圣杯"或具有同等力量的事物作为召唤媒介——人类无法仅凭自身力量召唤英灵 |

### 二、灵体与实体

从者拥有两种存在形态：

| 形态 | 特征 | 用途 |
|------|------|------|
| 灵体 | 不依赖物质构造；可穿越墙壁；物理攻击无效；魔力消耗低；难以被敌人发现 | 隐蔽移动、节省魔力、日常随行 |
| 实体 | 由魔力构成的肉体；可受物理干涉也可干涉物质世界；可进行战斗 | 战斗、与物质世界交互 |

从者可自由切换两种形态。灵体形态下无法攻击敌人，因此战斗时必须实体化。

### 三、击败从者的方法

从者现界时首先获得**灵核**——这是维持其存在的核心。

| 机制 | 描述 |
|------|------|
| 灵核弱化 | "耗费大量魔力"和"肉体受损伤"都会导致灵核弱化 |
| 灵核毁坏 | 灵核被破坏后，英灵无法维持现界，从者消亡 |
| 弱点部位 | 心脏和头部与灵核直接连结，受到攻击时灵核会遭受重创 |
| 有效手段 | 强大魔力、诅咒、宝具造成的伤害对灵核尤其有效 |

战斗中的核心策略：用尽可能少的魔力消耗给予敌方从者尽可能大的灵核伤害——因此情报收集和攻击弱点至关重要。

### 四、职阶系统（Class System）

#### 4.1 七大标准职阶

从者系统将英灵灌入预设的"职阶"框架。英灵本身没有职阶，职阶是从者独有的特征。

**三骑士**（上位职阶，要求苛刻）：

| 职阶 | 名称 | 特征要求 | 战斗特长 | 职阶能力 |
|------|------|---------|---------|---------|
| Saber | 剑士 | 与"剑之骑士"相称的传说；魔力外能力值最高 | 瞬间攻击力优秀 | 对魔力、骑乘 |
| Lancer | 枪兵 | 苛刻程度仅次于 Saber；全体能力值优秀，敏捷尤高 | 活用枪击范围和速度的"一击脱离"战法 | 对魔力 |
| Archer | 弓兵 | 具有强力射击武器或相关特殊能力 | 宝具强大；适合侦察 | 对魔力、单独行动 |

**四骑兵**（下位职阶）：

| 职阶 | 名称 | 特征要求 | 战斗特长 | 职阶能力 |
|------|------|---------|---------|---------|
| Rider | 骑兵 | 与乘坐物有渊源的传说 | 以坐骑性能补救能力值的不足 | 对魔力、骑乘 |
| Caster | 魔术师 | 魔术能力达最高等级 | 魔术战；战斗能力较低，被评为最弱职阶 | 阵地建造、道具作成 |
| Assassin | 暗杀者 | 冬木圣杯战争中限定为"历代哈桑·萨巴赫" | 气息遮断；以隐秘行动弥补低能力值 | 气息遮断 |
| Berserker | 狂战士 | "曾在战斗中疯狂"的英雄 | 以失去理性为代价获得超越英灵性能的强化 | 狂化 |

#### 4.2 额外职阶

在特殊情况或变体圣杯战争中出现的非标准职阶：

| 职阶 | 名称 | 出现场景 | 特征 |
|------|------|---------|------|
| Ruler | 裁定者 | Fate/Apocrypha | 圣杯战争的仲裁者，拥有对所有从者使用令咒的权限 |
| Avenger | 复仇者 | 第三次冬木圣杯战争 | 由爱因兹贝伦违规召唤，导致大圣杯污染 |
| Shielder | 盾兵 | Fate/Grand Order | 防御特化型从者 |
| Foreigner | 降临者 | Fate/Grand Order | 与外神有关联的从者 |
| Alter Ego | 分身 | Fate/Extra CCC | 从某一存在分裂而来的从者 |
| Moon Cancer | — | Fate/Extra CCC | Moon Cell 系统中的异常存在 |

#### 4.3 职阶分配规则

- 英灵根据其特性决定符合的职阶。
- 若同一英灵适合多个职阶，由 Master 与该英灵的相性决定最终职阶。
- 御三家等深知系统规则者可有预谋地干涉分配（如刻意将强者召唤为 Berserker）。

### 五、能力参数系统

#### 5.1 六大能力值

| 能力值 | 含义 |
|--------|------|
| 筋力（Strength） | 肉体力量的强度 |
| 耐久（Endurance） | 能承受伤害的程度 |
| 敏捷（Agility） | 反应速度和敏捷性 |
| 魔力（Mana） | 能操纵的魔力量 |
| 幸运（Luck） | 运气的好坏，影响关键时刻的判定 |
| 宝具（Noble Phantasm） | 拥有的宝具的强度 |

#### 5.2 等级体系

| 等级 | 数值 | 说明 |
|------|------|------|
| E | 10 | 最低标准 |
| D | 20 | 低于一般水平 |
| C | 30 | 一般水平 |
| B | 40 | 高于一般水平 |
| A | 50 | 最高标准 |
| EX | — | 别格——强到没有比较意义 |
| + | ×2 | 一瞬间将数值倍化（C+ = 瞬间 60） |
| ++ | ×3 | 一瞬间将数值三倍化 |

#### 5.3 影响面板的三大要素

| 要素 | 影响方式 |
|------|---------|
| 土地 | 距离英灵传说所在的文化圈越近，强度和装备越接近传说描述 |
| 知名度 | 英灵在当地的知名度越高，性能越强 |
| Master 的魔力 | Master 魔力越强大，从者越能发挥本来的性能 |

### 六、职阶能力详解

| 能力名 | 对应职阶 | 效果 |
|--------|---------|------|
| 对魔力 | Saber、Lancer、Archer、Rider | 将同等级以下的魔术无效化；B 以上可抵御大魔术 |
| 骑乘 | Saber、Rider | 驾驭乘坐物；A+ 以上可驾驭幻兽、神兽 |
| 单独行动 | Archer | 失去 Master 后可继续现界（C 级 1 天，B 级 2 天，A 级可无限现界） |
| 阵地建造 | Caster | 制造有利于自己的魔术阵地（A 级可建造"神殿"级阵地） |
| 道具作成 | Caster | 制作带有魔力的器具和魔术礼装（A 级可制作拟似不死药） |
| 气息遮断 | Assassin | 隐藏从者气息（A+ 级几乎不可能被发现） |
| 狂化 | Berserker | 以理性为代价提升能力值（B 以上全能力值上升） |

### 七、固有技能（保有技能）

区别于职阶能力——固有技能是"英灵本身所持有的技能"，来源于该英灵的传说和生前技术。

- 例如：直觉（预知危险）、战斗续行（重伤后继续战斗）、魔力放出（将魔力附着于武器增幅攻击力）、千里眼（远距离视觉和洞察力）等。
- 固有技能的种类和等级因英灵个体而异，是从者之间差异化的重要来源。

### 八、宝具系统（Noble Phantasm）

#### 8.1 宝具的定义

宝具是英灵传说中被提及和称颂的**最强武装**——以人类的幻想为骨架制作的武装。宝具拥有现代魔术师望尘莫及的力量，一旦使用便发挥传说般的效果。

宝具可以是：武器、盾牌、装饰品、坐骑、招式本身、固有结界，甚至是概念性的能力。

#### 8.2 宝具的发动

- **真名解放**：从者高呼宝具真名并注入魔力来发动全力。
- 真名解放暴露真名的风险，因此从者通常将宝具作为底牌保留。

#### 8.3 宝具的分类

| 类别 | 效果范围 | 特征 |
|------|---------|------|
| 对人宝具 | 单体 | 对有限目标给予效果，魔力消耗少，一对一效率极高 |
| 对军宝具 | 数十~数百人 | 广范围攻击，扩散庞大能量一举轰飞敌军 |
| 对城宝具 | 城塞级 | 可击溃坚固构造物，威力与前两者有悬殊差距 |
| 对界宝具 | 世界级 | 拥有切开世界等传说级效果，极为稀有 |
| 结界宝具 | 特定空间 | 构筑结界但不用于攻击的宝具（如 Avalon 封杀所有干涉） |
| 对魔术宝具 | 魔术体系 | 以魔术构造物为对象，将其返回形成前状态 |

#### 8.4 宝具的战略意义

- 宝具是从者战力的**决定性因素**——职阶强弱只是基础，宝具的适配往往决定实际战斗的胜负。
- 宝具暴露真名的风险使得"宝具情报"成为最高价值的情报。
- 宝具使用需要大量魔力，过度使用会加速灵核弱化。

### 九、从者的属性（阵营）

从者拥有表现其精神性的"属性"，由两个维度组合：

| 维度 | 分类 |
|------|------|
| 行动方针 | 秩序（Lawful）、中立（Neutral）、混沌（Chaotic） |
| 性格 | 善（Good）、中庸（Neutral）、恶（Evil） |

方针不同的从者容易产生意见分歧；性格不同的从者可能不会产生大冲突。

## 第二部分：Master（御主）

### 一、Master 的定义与资格

| 属性 | 描述 |
|------|------|
| 定义 | 被圣杯承认、成功召唤出从者的人 |
| 基本条件 | 拥有魔术回路（通常为魔术师，但不绝对） |
| 资格凭证 | 令咒——三划刻印刻于肉体上 |
| 核心功能 | 作为从者现世的"依凭"（锚点）+ 供应魔力的源泉 |

### 二、Master 的核心能力

#### 2.1 魔力供给

| 方面 | 描述 |
|------|------|
| 供魔路径 | 召唤成功时自动建立 Master-Servant 之间的魔力通道 |
| 状态感知 | 通过供魔路径，双方可一定程度感知彼此的位置和状态 |
| 供魔影响 | Master 魔力强大 → 从者接近传说本来的强度；Master 魔力不足 → 从者性能下降 |
| 供魔方式 | 通常为被动的自动传输；特殊情况可主动大量输送（消耗 Master 体力） |

#### 2.2 令咒使用

（详见《圣杯战争的核心规则》第四章）

Master 的令咒是控制从者的唯一强制手段，同时也是 Master 在战场上的"救命手段"——可用令咒让从者瞬移至身边保护自己，或强制从者全力一击决胜。

#### 2.3 魔术战斗能力

Master 本身通常也是魔术师，具有一定的战斗能力：
- 结界术：设置防护或感知入侵。
- 强化魔术：强化自身肉体或物品。
- 特定家传魔术：各家族传承的专精魔术。

但 Master 的战斗力远不及从者，直接面对从者基本毫无胜算。Master 的价值在于战略指挥和资源管理。

### 三、Master 的选定机制

| 优先级 | 条件 |
|--------|------|
| 最高 | 内心有强烈要实现的愿望（含潜意识） |
| 较高 | 与圣杯有因果联系的人（御三家成员等） |
| 中等 | 魔术资质优秀的魔术师 |
| 较低 | 没有强烈愿望的魔术师 |
| 最低 | 没有魔术回路的普通人（极端例外情况下可能被选中） |

### 四、Master 与 Servant 的关系

#### 4.1 关系的双重性

Master-Servant 关系本质上是一种**充满张力的合作**：

| 合作面 | 对抗面 |
|--------|--------|
| 共同的胜利目标 | 各自可能有不同的愿望和价值观 |
| Master 提供魔力和依凭 | Master 能力远弱于从者，需要令咒维持控制 |
| 从者提供战斗力 | 从者可能不认同 Master 的战略选择 |
| 信息共享的需要 | 从者可能隐瞒自身弱点或过去 |

#### 4.2 关系类型谱系

根据 Fate 系列各作品归纳的典型 Master-Servant 关系：

| 类型 | 特征 | 代表案例 |
|------|------|---------|
| 战友型 | 互相信赖、目标一致、配合默契 | 韦伯 & 征服王 |
| 主仆型 | Master 明确指挥、Servant 忠实执行 | 远坂时臣 & 吉尔伽美什（名义上） |
| 工具型 | Master 将从者视为纯工具、缺乏尊重 | 间桐家的刻印虫系统 |
| 对立型 | Master-Servant 价值观严重冲突 | 卫宫切嗣 & 阿尔托莉雅 |
| 共情型 | 双方在战争中逐渐理解并影响彼此 | 卫宫士郎 & 阿尔托莉雅 |
| 寄生型 | 一方完全依附另一方 | 失去 Master 的从者寻找新宿主 |

## 第三部分：圣杯战争中的行动模式

### 一、Servant 的典型行动

#### 1.1 侦察与情报收集

| 行动 | 描述 |
|------|------|
| 灵体巡逻 | 以灵体形态在城市中移动，不被普通人感知，探查其他从者的气息 |
| 交战试探 | 在初期遭遇战中刻意隐藏宝具和真名，用基础战斗能力试探对方实力 |
| 真名推理 | 根据对手的战斗风格、武器特征、使用的技能推测其真名 |

#### 1.2 战斗行动

| 行动 | 描述 |
|------|------|
| 正面对决 | 从者间的直接武力对抗，主要发生在夜间 |
| 宝具解放 | 高呼真名释放宝具全力，通常作为决胜或逆转的底牌 |
| 阵地防御 | Caster 类从者在据点构筑魔术阵地，利用主场优势迎击来犯者 |
| 奇袭暗杀 | Assassin 类从者利用气息遮断潜入目标附近发动突袭 |
| 坐骑突击 | Rider 类从者利用坐骑的机动性进行高速冲击战 |
| 狂暴突进 | Berserker 类从者依靠狂化后的压倒性力量进行不顾一切的攻击 |

#### 1.3 防御与保护

| 行动 | 描述 |
|------|------|
| Master 护卫 | 从者的首要任务是保护 Master 的安全——Master 死亡则从者失去依凭 |
| 紧急撤退 | 通过令咒的空间转移或自身能力迅速脱离战场 |
| 灵体回避 | 在不利局面下切换为灵体形态回避攻击（但持续灵体化无法反击） |

### 二、Master 的典型行动

#### 2.1 战略层面

| 行动 | 描述 |
|------|------|
| 情报收集 | 通过使魔（familiars）、魔术监控、社会关系网络收集敌方信息 |
| 敌方分析 | 推断敌方从者真名、弱点和宝具类型 |
| 同盟缔结 | 与其他 Master 建立临时同盟以对抗共同威胁 |
| 同盟破弃 | 在利用完同盟价值后背叛对方 |
| 资源管理 | 管理自身魔力消耗、令咒使用时机、补给品等 |
| 战场选择 | 选择有利于己方从者的战斗地点和时机 |

#### 2.2 战术层面

| 行动 | 描述 |
|------|------|
| 令咒指令 | 在关键时刻使用令咒实现逆转（瞬移、强化、强制行动等） |
| 魔术支援 | 以自身魔术为从者提供辅助（结界、强化、治疗等） |
| 诱敌陷阱 | 设置陷阱或诱饵引导敌方进入不利局面 |
| 袭击对方 Master | 直接针对实力较弱的敌方 Master 而非其从者 |
| 结界布置 | 在据点或要道设置感知结界和防护结界 |

#### 2.3 社会层面

| 行动 | 描述 |
|------|------|
| 日常伪装 | 在非战斗时维持普通人的社会身份 |
| 资源筹备 | 准备魔术道具、触媒、补给品、安全屋等 |
| 与监督者交涉 | 在需要时联络监督者，了解战况或申请庇护 |
| 善后处理 | 清除战斗痕迹，对目击者进行记忆操作 |

### 三、Master-Servant 的协同战术

| 战术 | 描述 |
|------|------|
| 诱杀战术 | Master 佯装暴露诱敌，从者伏击来犯的敌方从者 |
| 分头行动 | Master 进行情报收集或社交，从者独立执行战斗任务 |
| Master 狙击 | 从者牵制敌方从者，Master 绕路袭击敌方 Master |
| 据点防守 | Master 在据点内维持结界，从者在结界加持下迎战来敌 |
| 多方协调 | 与同盟组协同，形成夹击或分工配合 |
| 令咒+宝具联动 | 在宝具发动的关键瞬间用令咒追加强化，实现超规格的一击 |

### 四、从者之间的典型对抗模式

| 对抗模式 | 描述 |
|----------|------|
| 近战白刃战 | Saber vs Lancer 等近战职阶的正面武艺对决 |
| 远近博弈 | Archer 远程牵制 vs 近战职阶的接近战 |
| 概念对冲 | 宝具效果之间的概念性对抗（如矛与盾的悖论） |
| 阵地攻防 | Caster 据守阵地，其他从者强攻突破 |
| 暗杀博弈 | Assassin 暗中觊觎，其他从者需要保持警觉 |
| 狂暴压制 | Berserker 以压倒性暴力突进，其他从者需要智取 |
| 固有结界 | 从者展开固有结界改变战场规则，强制进入有利局面 |

## 附录：从者的系统性弱点与制约

| 制约 | 描述 |
|------|------|
| 依赖 Master | 失去 Master 后除 Archer 外大多无法长期维持现界 |
