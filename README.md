# FateClaw

AIvilization Plus 游戏设计文档仓库。项目经历两个阶段：

1. **FateClaw（已封存）**：Agent 驱动的圣杯战争模拟，Master + LLM Servant 在冬木市对战。因复杂度过高，MVP v1 已于 2026-03-25 封存。
2. **TinyMB（当前方向）**：Tiny Mount & Blade，以极简骑砍规则验证多 Agent 沙盒世界，基于 Trumen 框架运行。

> 本仓库仅含设计文档，实现代码位于 Trumen 工程。
