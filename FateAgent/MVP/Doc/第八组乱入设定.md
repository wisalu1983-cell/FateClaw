# 第八组乱入设定（MVP v1）

> **文档用途**：定义玩家作为第八组 Master 的参战框架、可选 Servant 候选列表及乱入规则。
>
> **核心设定**：第五次冬木市圣杯战争的七个标准职阶已被占满。玩家的第八组是通过异常途径参战的"乱入者"。

---

## 一、第八组 Master（玩家）

### 基础框架

| 属性 | 设定 |
|------|------|
| 身份 | 由系统预设或玩家在开局阶段选择的背景（v1 可简化为统一背景） |
| 参战方式 | 异常参战——非大圣杯正式选定的 Master，而是通过某种异常途径获得令咒和从者 |
| 令咒 | 经典 3 划 |
| 魔术能力 | 中等（足以维持从者现界，但不能自身参战） |
| 初始信息 | 知道自己参加的是圣杯战争；知道基本规则；不知道其他参战者的具体身份 |
| 初始据点 | 无固定据点，需要在战争初期自行确定落脚点 |
| 目标 | 完全由玩家自主决定。系统不预设"你要赢"或"你要扶持谁"——唯一的世界规则是最终只有一组获胜 |

### 异常参战的叙事理由（v1 简化版）

第五次冬木圣杯战争中，大圣杯因第三次战争的污染（安哥拉曼纽）存在系统性异常。这种异常在从者召唤阶段产生了一个"溢出"——额外分配了一组令咒给本不该被选中的候选人。玩家就是这个溢出的受益者。

这一设定的好处：
- 解释了为什么会有第八组（圣杯异常导致的溢出）
- 和原作大圣杯污染的伏笔保持一致
- 不需要创造全新的设定架构
- 为后续版本扩展暗线（大圣杯污染的更多影响）留下接口

### 被其他组察觉的规则

| 触发条件 | 结果 |
|----------|------|
| 从者在某地点实体化行动 | 该地点及相邻地点的从者有概率感知到异常气息 |
| 与其他组发生直接接触（对话、交战） | 对方确认第八组存在 |
| 在高人流地点行动 | 提高被所有组发现的概率 |
| 主动前往冬木教会拜访监督者 | 绮礼会知道第八组存在（但可能不立即公开） |
| Lancer 的全城巡逻 | Lancer 有概率在巡逻中发现第八组从者的气息 |

### 其他组对第八组的初始态度推演

| 组别 | 可能反应 |
|------|---------|
| 士郎组 | 困惑但不会主动敌对。如果玩家表现出善意，有结盟可能 |
| 凛组 | 警惕且理性。会优先调查第八组的实力和意图。有条件合作的可能 |
| 慎二组 | 视第八组为新威胁或可利用的对象。可能试图拉拢或排挤 |
| 伊莉雅组 | 对新变量保持警觉。如果判断为弱者可能直接出手试探 |
| Caster 组 | 视为潜在的从者掠夺目标（Rule Breaker） |
| 绮礼/Lancer | 绮礼会迅速获知第八组的存在，并评估其对自己计划的影响。可能派 Lancer 试探 |
| 吉尔伽美什 | 如果注意到，会以旁观者态度观察这个"不速之客" |

## 二、可选 Servant 候选列表

> 玩家在开局阶段从以下 7 名从者中选择 1 名作为搭档。每名从者的职阶、能力、性格和与原作角色的互动潜力均不相同，会显著改变玩家的策略空间和体验走向。

`攻击型` `情报隐蔽` `正面突破` `对Saber特攻` `性格冲突高`

#### 性格与互动模式

- 温柔、正义、坚定。不是天真的圣人——经历过火刑的她对人性的黑暗有深刻理解
- 对 Master 的态度：尊重且配合。会在 Master 偏离正道时提出异议，但最终尊重 Master 的决定。不会因立场不同就背叛
- 令咒敏感度：完全接受——她理解契约的含义。但如果令咒被用于明显不义的行为，会在心中留下阴影
- 独特优势：**真名看破 + 对全从者令咒是两张王牌**。前者让玩家在情报战中拥有压倒性优势，后者给予了对任何从者的紧急干预能力

#### 与第五次战争角色的互动潜力

| 吉尔伽美什 | 在 Fate/Zero 和 FGO 中，贞德与吉尔·德·雷（第四次 Caster）有关联。虽然第五次战争中没有吉尔·德·雷，但贞德的存在可能引起吉尔伽美什的兴趣 |

#### 玩法标签

`情报外交型` `防御支援` `裁判权能` `战略指挥` `**唯一的额外职阶选项**`

### 候选 7：坂田金时（Berserker）

#### 基本信息

| 属性 | 详情 |
|------|------|
| 真名 | 坂田金时（Sakata Kintoki） |
| 传说原型 | 平安时代源赖光四天王之首，童话"金太郎"的原型。自幼在足柄山中由山姥养大，拥有怪力。成年后成为源赖光麾下最强的武士，参与讨伐酒吞童子等妖魔 |
| 职阶 | Berserker |
| 属性 | 秩序·善 |

#### 能力参数

| 筋力 | 耐久 | 敏捷 | 魔力 | 幸运 | 宝具 |
|------|------|------|------|------|------|
| A+ | B | B | C | C | A |

#### 职阶能力

| 能力 | 等级 | 效果 |
|------|------|------|
| 狂化 | E | 最低等级的狂化。仅有微弱的参数提升，但作为代价——**完全保留了理性和语言能力**。这是极为罕见的 Berserker 特性 |
| 对魔力 | D | 一节以下的咏唱魔术无效化 |

#### 固有技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 天性の肉体 | A | 天生的强壮肉体。对筋力系的判定有额外加成 |
| 动物会话 | C | 能够与动物沟通。在侦察时可以利用 |
| 自然の嬰儿 | A | 在大自然中感知力大幅提升 |

#### 宝具

| 宝具 | 等级 | 种类 | 效果 |
|------|------|------|------|
| 黄金衝撃（Golden Spark） | A | 对人/对军宝具 | 以雷电之斧释放的黄金色冲击。瞬间爆发力极大，可以在一击中决定胜负。也可作为远程攻击释放 |

#### 性格与互动模式

- 豪爽、单纯、热血。说话像现代不良少年/机车族——"GOLDEN！""那可太酷了！"
- **是所有候选中唯一"能正常对话的 Berserker"**——狂化等级 E 意味着他几乎和正常从者没有区别
- 对 Master 的态度：直来直去，如果认可 Master 就全力以赴。对"不爽的命令"会直接表达不满但不会违抗
- 令咒敏感度：不太在意——"既然签了约就是伙伴"
- 独特优势：筋力 A+ 使他在纯粹的物理对抗中仅次于 Berserker（赫拉克勒斯）。保留语言能力意味着他可以自主判断、自主侦察、自主协商

#### 与第五次战争角色的互动潜力

| 对象 | 互动内容 |

| 伊莉雅 | 金时的单纯性格可能与伊莉雅产生有趣的互动——伊莉雅可能在他面前展现更真实的一面 |

## 四、从者选择对玩法的影响

不同从者选择会显著改变玩家的策略空间：

| 策略方向 | 最适合的从者 |
|----------|-------------|
| 正面武力碾压 | 斯卡哈、坂田金时 |

| 外交/结盟优先 | 伊斯坎达尔、贞德 |
| 暗杀/定点清除 | 杰克 |
| 据点防御/消耗战 | 玉藻前 |
| 制造叙事冲突 | 莫德雷德（对Saber）、斯卡哈（对Lancer）、伊斯坎达尔（对吉尔伽美什） |
| 对特定敌人有特攻 | 莫德雷德（对阿尔托莉雅）、杰克（对女性Master/夜间）、斯卡哈（对神性） |
