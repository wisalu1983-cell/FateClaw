# 骑砍经济、贸易与聚落繁荣度

> **文档用途**：详细记录骑砍的经济系统运转机制、贸易商品数据、聚落繁荣度和队伍管理数值，作为 TinyMB 数值骨架设计（Phase 1 步骤 1.6）的输入材料。
>
> **数据来源**：Mount & Blade Wiki (mountandblade.fandom.com)、StrategyWiki (strategywiki.org)、Steam 社区指南。以战团为主，标注霸主差异。

---

## 一、经济系统总览

### 1.1 经济的宏观结构

骑砍的经济系统是一个**持续运转的、受战争影响的**供需模型：

```
村庄产出原材料 → 通过贸易路线/商队流入城镇 → 城镇加工销售
        ↑                                            ↓
    村庄繁荣度 ←—————— 安全巡逻 / 被劫掠 ——————→ 城镇繁荣度
```

战争破坏这个循环：劫掠村庄 → 原材料断供 → 城镇物价上涨 → 繁荣度下降。

### 1.2 经济对势力博弈的影响

经济财富影响势力均衡几乎与领土同等重要——它决定势力能否负担得起军事行动的成本。

## 二、贸易系统

### 2.1 供需机制

- **村庄**：产出原材料（皮毛、谷物、羊毛、鱼、牛肉），价格低于城镇
- **城镇**：消费原材料，出售加工品（工具、干肉、面粉），价格高于村庄
- 贸易是繁荣的关键：当贸易畅通时，商品充足、物价合理、人口增长；当贸易中断时，工人逃离、经济停滞

### 2.3 贸易策略规律

- **买入时机**：在基础价格 50% 以下买入通常有利润
- **高价值商品优先**：基础价格 > 100 的商品利润空间更大
- **最赚钱的商品**：天鹅绒、香料、油、工具、盐、铁
- **村庄→城镇套利**：村庄的原材料价格低于城镇，加工品价格高于城镇

### 2.4 商队系统

NPC 商队在城镇之间自动流动，是城镇间物资流通的载体。商队可以被攻击（损害与对应势力的关系 -5）。战争期间商队路线被打断，影响沿线城镇经济。

## 三、聚落繁荣度（Prosperity）

### 3.1 概念

每个聚落（城镇/城堡/村庄）都有一个**繁荣度**变量，代表聚落的经济健康状态。

### 3.2 繁荣度的影响

| 受影响项 | 高繁荣度 | 低繁荣度 |
|---|---|---|
| 税收 | 更高 | 更低 |
| 物价 | 更合理 | 更高/短缺 |
| 可招募兵种质量 | 更高级 | 只有基础兵 |
| 可招募数量 | 更多 | 更少 |
| 人口 | 增长 | 流失 |

### 3.3 影响繁荣度的因素

| 因素 | 影响方向 | 说明 |
|---|---|---|
| 贸易畅通 | ↑ | 商队正常运行、商品流通 |
| 巡逻安全 | ↑ | 附近无强盗威胁 |
| 和平时期 | ↑ | 非战争状态下缓慢恢复 |
| 村庄被劫掠 | **↓ 暴跌** | 降至"极穷"，需要很长时间恢复 |
| 城镇被攻陷 | **↓ -20** | 一次攻城事件直接扣除 |
| 战争持续 | ↓ | 贸易路线中断、商队被劫 |

### 3.4 繁荣度的经济循环

```
繁荣度高 → 税收高 → 领主有钱扩军 → 军事力量强 → 保卫领地
    ↑                                                    ↓
繁荣度恢复 ← 和平/巡逻安全 ← 军事优势 ← 战争胜利 ←←←←←
    ↓
繁荣度低 → 税收低 → 领主缺钱 → 军事力量弱 → 领地被劫掠
    ↑                                              ↓
    ←←←←←←←← 繁荣度进一步下降 ←←←←←←←←←←←←←←←←
```

这形成了一个**正反馈循环**——强者愈强、弱者愈弱。是骑砍"势力兴衰"的经济驱动力。

## 四、队伍管理数值

### 4.1 士气系统

**基础士气**：50 点

**增加士气**：

| 来源 | 说明 |
|---|---|
| Leadership 技能 | 每级提供正面加成 |
| 食物种类多样性 | 每种不同类型的食物额外加成 |
| 赢得战斗 | 短期提升 |
| 劫掠村庄 | 短期提升（但损害荣誉） |
| 赢得比武 | 短期提升 |
| 完成任务 | 短期提升 |

**降低士气**：

| 来源 | 说明 |
|---|---|
| 队伍规模 | 人越多惩罚越大——大军团的凝聚力问题 |
| 饥饿 | 无食物时大幅降低 |
| 战败 | 短期下降 |
| 强制招募俘虏 | 短期下降 |
| 一天内战斗过多 | 每天 1~2 场以上开始下降 |
| 多势力混编 | 不同势力的兵在一起会降低士气 |
| 任务失败 | 短期下降 |
| 欠薪 | 大幅下降 |
| 误伤友军 | 下降 |

### 4.2 士气的影响

| 士气状态 | 影响 |
|---|---|
| 高士气 | 战斗更积极、行军速度更快 |
| 正常士气 | 正常表现 |
| 低士气 | 行军变慢、战斗消极 |
| 极低士气 | 开始逃兵（部队自动减少） |

### 4.3 队伍规模

队伍上限受以下因素影响：
- **声望**：每 25 声望 +1 人
- **Charisma 属性**：提供基础上限
- **Leadership 技能**：进一步增加上限
- **家族等级（霸主）**：每升一级 +15 人

### 4.4 队伍移动速度

影响因素：

| 因素 | 效果方向 |
|---|---|
| Pathfinding 技能 | ↑ |
| 骑乘部队比例 | ↑ |
| 高士气 | ↑ |
| 小队伍 | ↑ |
| 白天 | ↑ |
| 携带囚犯 | ↓ |
| 大队伍 | ↓ |
| 货物重量 | ↓ |
| 恶劣天气 | ↓ |
| 夜间 | ↓ |
| 森林等复杂地形 | ↓ |

备用马可以部分抵消货物重量的速度惩罚。

## 五、产业系统

### 5.1 工坊/产业

玩家可在城镇购买产业用地开设工坊：

| 常见工坊类型 | 原材料 | 产出 | 收益特征 |
|---|---|---|---|
| 酿酒厂 | 谷物 | 麦酒 | 稳定收入 |
| 铁匠铺 | 铁 | 工具/武器 | 高收入但依赖铁矿供给 |
| 纺织厂 | 羊毛/亚麻 | 布料/天鹅绒 | 中等收入 |
| 制革厂 | 兽皮 | 皮革制品 | 中等收入 |
| 榨油坊 | 橄榄 | 油 | 依赖橄榄产地 |

**关键**：工坊利润取决于原材料的获取成本，而原材料价格受附近村庄繁荣度和贸易路线影响。

### 5.2 领地税收

| 领地类型 | 税收水平 | 影响因素 |
|---|---|---|
| 城镇 | 最高 | 繁荣度、人口、贸易量 |
| 城堡 | 中等 | 附属村庄贡献 |
| 村庄 | 最低 | 繁荣度、是否被劫掠 |

税收每周自动结算，是拥有领地后最稳定的收入来源。

## 六、对极简化经济设计的启示

### 6.1 必须保留的核心机制

| 机制 | 理由 |
|---|---|
| 军饷压力 | 驱动持续行动的核心经济约束 |
| 领地税收 | 封臣身份的核心经济回报 |
| 繁荣度正反馈循环 | 势力兴衰的经济驱动力 |
| 战争对经济的破坏 | 军事行动有经济后果 |

### 6.2 可以大幅简化的部分

| 原版机制 | 极简化方向 |
|---|---|
| 37 种贸易商品 | 缩减为 3~5 种抽象资源类型 |
| 详细的供需价格波动 | 简化为"聚落繁荣度影响收入" |
| 工坊系统 | 移除，合并到领地税收中 |
| 商队系统 | 移除，用"贸易安全度"抽象替代 |
| 队伍速度的多因素计算 | 简化为"队伍规模影响移动时间" |

### 6.3 极简经济的骨架建议

```
收入 = 领地税收（受繁荣度影响） + 任务报酬 + 战利品
支出 = 军饷（受队伍规模和兵种等级影响）

繁荣度 ↑ ← 和平 + 安全巡逻
繁荣度 ↓ ← 被劫掠 + 战争

经济压力 = 支出 > 收入 → 需要行动（做任务/交易/劫掠/扩张领地）
```
