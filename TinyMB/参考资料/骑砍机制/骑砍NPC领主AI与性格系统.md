# 骑砍 NPC 领主 AI 与性格系统

> **文档用途**：详细记录骑砍中 NPC 领主的 AI 行为逻辑、性格类型分类及其行为影响，作为 TinyMB 极简化 NPC 自主行为设计（Phase 1 步骤 1.7）和 NPC Agent 方案决策（Q-06）的输入材料。
>
> **数据来源**：Mount & Blade Wiki (mountandblade.fandom.com)、Steam 社区隐藏机制指南、TaleWorlds 官方开发日志。以战团为主，标注霸主差异。

---

## 一、NPC 领主的行为模式

骑砍中的 NPC 领主独立运行自己的行为循环，与玩家使用同一套世界规则。

### 1.1 领主的六种基本行为

| 行为模式 | 说明 | 触发条件 |
|---|---|---|
| **补给（Resupply）** | 前往己方聚落补充士兵和物资 | 部队损耗后 |
| **巡逻（Patrol）** | 在己方领地附近巡逻，驱赶强盗 | 无紧急威胁时的默认行为 |
| **防御（Defense）** | 赶赴受威胁的己方聚落进行防守 | 己方聚落被攻击/围困 |
| **进攻（Offense）** | 对敌方聚落发动劫掠或攻城 | 战争状态下的主动出击 |
| **社交（Social）** | 参加宴会、拜访其他领主、外交活动 | 宴会邀请、外交需求 |
| **经济（Economic）** | 贩卖战俘、处理领地事务 | 有战俘需要处理或领地需要管理 |

### 1.2 Campaign AI 难度对领主行为的影响

战团中有三个 Campaign AI 难度等级，直接影响 NPC 领主的行为能力：

| 能力 | Poor（弱） | Average（中） | Good（强） |
|---|---|---|---|
| 战败恢复速度 | 缓慢，难以重建 | 中等 | 约 1 周即可完全恢复 |
| 招募成本 | 高 | 中 | 低（快速扩军） |
| 对玩家的敌意偏向 | 低 | 中 | 高——更不愿跟随玩家元帅、更倾向对玩家宣战 |
| 队伍规模上限 | 同（由玩家等级决定） | 同 | 同，但更快达到上限 |
| 攻击玩家封地倾向 | 低 | 中 | 高 |
| 加入玩家王国意愿 | 较高 | 中 | 较低 |

**关键洞察**：Campaign AI 并非让 NPC 更"聪明"，而是通过调整经济参数（招募成本、恢复速度）间接影响行为。高难度 = NPC 资源更充裕 = 行为更激进。

## 二、领主性格类型系统

### 2.1 八种原始性格类型

战团中共有 8 种性格类型，可归为三大阵营、四大倾向：

| 阵营 | 性格类型 | 倾向 | 忠诚度 | 叛变概率 |
|---|---|---|---|---|
| **正直阵营** | Upstanding（正直） | 荣誉导向 | 极高 | 极低 |
| **正直阵营** | Good-natured（善良） | 荣誉导向 | 高 | 低 |
| **中性阵营** | Martial（好战） | 武力导向 | 较高 | 较低 |
| **中性阵营** | Cunning（狡猾） | 利益导向 | 中 | 中 |
| **中性阵营** | Pitiless（冷酷） | 利益导向 | 中低 | 中高 |
| **中性阵营** | Bad-tempered（暴躁） | 利益导向 | 低 | 高 |
| **不正直阵营** | Sadistic（残忍） | 自利导向 | 极低 | 极高 |
| **不正直阵营** | Debauched（堕落） | 自利导向 | 极低 | 极高 |

### 2.2 性格对具体行为的影响

#### 被俘后的反应

| 性格 | 被释放后关系变化 | 被俘时的态度 |
|---|---|---|
| Upstanding（正直） | +1 ~ +2 | 以礼相待，请求骑士礼遇 |
| Good-natured（善良） | +1 ~ +2 | 祝贺你的胜利 |
| Martial（好战） | +5 | 要求下令投降 |
| Cunning（狡猾） | 0（无变化） | 冷静分析局势 |
| Pitiless（冷酷） | 未明确 | — |
| Bad-tempered（暴躁） | 未明确 | 诅咒这一天 |
| Sadistic（残忍） | **-15** | 被冒犯，愤怒 |
| Debauched（堕落） | 未明确 | 不甘地承认失败 |

**关键差异**：释放正直/善良型领主获得正面关系；释放残忍型领主反而大幅损失关系（-15）。

#### 对分封的态度

| 性格 | 分封态度 |
|---|---|
| Upstanding | 不在意分封结果 |
| Martial | 轻微关注 |
| Cunning | 关注公平性 |
| Sadistic/Debauched | 自私——期望自己获封 |

#### 战争行为倾向

| 性格倾向 | 战争行为 |
|---|---|
| 正直/善良 | 偏好巡逻防御、主动救援友军，忠实响应召集 |
| 好战 | 偏好进攻、主动挑战敌军 |
| 狡猾 | 行为灵活，追求利益最大化 |
| 冷酷/暴躁 | 偏好劫掠，容易叛变 |
| 残忍/堕落 | 不守承诺，高度不可靠 |

#### 作为元帅的表现

| 性格 | 元帅表现 |
|---|---|
| 好战型 | 积极发动进攻，推进战役 |
| 厌战型 | 留在己方领土内，战役缺乏进展 |
| 正直型 | 平衡进攻和防御 |

### 2.3 性格识别方法

玩家可以通过以下方式判断领主性格：

1. **初次对话风格**：礼貌 = 好领主，侮辱/威胁 = 差领主
2. **私下交谈**：询问对国王和政治的看法
3. **被俘时的反应**：不同性格有不同的投降台词
4. **行为观察**：长期观察其战争/和平选择偏好

## 三、领主叛变机制

### 3.1 战团叛变条件

领主叛变受以下因素影响：

| 因素 | 影响方向 |
|---|---|
| 与当前国王的关系 | 关系越差越可能叛变 |
| 势力实力对比 | 己方势力越弱越可能叛变 |
| 领主性格 | 不正直型更容易叛变 |
| 玩家的 Right to Rule | 玩家 Right to Rule 越高，领主越可能被说服 |
| 与玩家的关系 | 高关系才能进入私下交谈劝说叛变 |

### 3.2 霸主叛变机制变化

| 差异 | 战团 | 霸主 |
|---|---|---|
| 叛变触发 | 主要通过玩家说服 | 可能自动叛变 |
| 成本 | 主要依赖关系和性格 | 需要大量金钱（无封地领主 30~40 万；有封地领主约 200 万） |
| 领地转移 | 叛变领主带走领地 | 叛变领主**不带**领地 |
| 性格影响 | 性格匹配度重要 | 高荣誉领主可能免费加入（共同荣誉特质） |

## 四、势力级 AI 决策

### 4.1 宣战与议和

**战团**：国王/元帅决定宣战，不经投票。

**霸主**：通过 Kingdom Diplomacy 投票系统：
- 提议宣战/议和需要消耗影响力
- 成功概率基于王国整体紧张度
- 所有封臣可投票，票权与消耗的影响力挂钩

### 4.2 元帅选举

- 在战团中，由国王任命
- 高声望、高荣誉、高关系的领主更可能被任命
- 元帅可以召集领主组成大军

### 4.3 分封决策

**战团**：国王独自决定分封。每次分封：
- 获封者 +10 关系
- **所有其他封臣** -2 关系
- 性格影响封臣对分封的反应强度

**霸主**：通过投票系统决定：
- 攻陷领地后，系统提名若干候选人
- 封臣投票，消耗影响力增加票权
- 投票结果决定最终归属

## 五、对极简化 NPC 设计的启示

### 5.1 性格系统的极简化方向

原始 8 种性格可归为 **3~4 种行为模式**：

| 极简性格 | 对应原始类型 | 核心行为特征 |
|---|---|---|
| **忠诚型** | Upstanding + Good-natured | 忠于誓言、偏好防御巡逻、关系稳定 |
| **好战型** | Martial | 偏好进攻、追求战斗荣耀 |
| **投机型** | Cunning + Pitiless | 灵活切换、追求利益最大化 |
| **不可靠型** | Bad-tempered + Sadistic + Debauched | 容易叛变、偏好劫掠 |

### 5.2 领主行为的 Agent 化映射

| 骑砍领主行为 | Agent 实现方式 |
|---|---|
| 评估当前状态 → 选择行为模式 | Agent 的 Observe → Think 循环 |
| 性格影响行为偏好 | Agent 的人格设定（Prompt 中的性格描述） |
| 基于关系和实力选择敌友 | Agent 读取关系数值做决策 |
| 响应外部事件（被攻击/被召集） | Agent 对环境事件的响应 |

### 5.3 核心决策问题（Q-06）

NPC 领主是否需要作为完整的 LLM Agent？

| 方案 | 优势 | 劣势 |
|---|---|---|
| **完整 Agent** | 行为最真实、可产生丰富社交事件 | LLM 调用成本最高 |
| **规则驱动 + 关键时刻 LLM** | 日常行为用规则（成本低），社交/外交用 LLM（质量高） | 需要定义"关键时刻"的边界 |
| **纯规则驱动** | 成本最低 | 行为模式固定，缺乏涌现性 |
