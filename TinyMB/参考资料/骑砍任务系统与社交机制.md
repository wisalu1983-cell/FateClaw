# 骑砍任务系统与社交机制

> **文档用途**：详细记录骑砍的任务系统分类、宴会系统和社交互动机制，作为 TinyMB 核心循环行为空间设计（Phase 1 步骤 1.5）和玩家引导设计（步骤 1.8）的输入材料。
>
> **数据来源**：Mount & Blade Wiki (mountandblade.fandom.com)、StrategyWiki (strategywiki.org)。以战团为主。

---

## 一、任务系统概述

战团中约有 **55 种任务**，由不同类型的 NPC 提供。任务是 Agent 行为多样性的重要来源，也是关系值获取的主要途径。

### 1.1 核心规则

| 规则 | 说明 |
|---|---|
| 任务来源 | 村长 / 行会长 / 领主 / 国王 / 女性贵族 / 大臣 |
| 同时接取上限 | 每个 NPC 只能接一个活跃任务 |
| 时间限制 | 大多数任务有时间限制（30~50 天不等） |
| 取消惩罚 | 可向任务发布者对话取消，但损失关系和荣誉 |
| 刷新机制 | 完成或过期后经过冷却期（10~20 天）才会刷新新任务 |

## 二、任务分类详表

### 2.1 村长任务（Village Elder）

仅在**村庄**中可接。村长任务聚焦乡村生存问题。

| 任务名 | 目标 | 奖励 | 关系影响 |
|---|---|---|---|
| **运送小麦** | 购买并运送小麦到指定村庄 | 金钱 | 村庄 +10 |
| **运送牛群** | 从其他村庄购买牛群赶回 | 金钱 | 村庄 +8 |
| **训练农民** | 在村庄中训练农民对抗强盗 | 金钱 + 荣誉选项 | 村庄 +3~11 |

**训练农民**任务的荣誉选项：击败强盗后可选择拒绝报酬（+1 荣誉）。

### 2.2 行会长任务（Guild Master）

仅在**城镇**中可接。行会长任务聚焦商业和民生。

| 任务名 | 目标 | 奖励 | 关系影响 |
|---|---|---|---|
| **驱赶牛群** | 将牛群从一个地方赶到另一个地方 | 金钱 | 城镇 +1 |
| **护送商队** | 护送 NPC 商队到达指定城镇 | 金钱 | 城镇 +1 |
| **运送酒水** | 运送麦酒到指定城镇 | 金钱 | 城镇 +1 |
| **剿灭强盗** | 消灭附近区域的强盗 | 金钱 | 城镇 +2 |
| **从强盗手中赎回女孩** | 击败强盗救出被绑架的女孩 | 金钱 | 城镇 +2 |
| **处理掠夺者** | 消灭城镇附近的掠夺者 | 金钱 | 城镇 +5 |
| **拯救城镇** | 解除城镇面临的强盗威胁 | 金钱 | 城镇 +0~14 |
| **说服领主议和** | 说服交战双方的领主达成和平 | 金钱 + 荣誉选项 | 城镇 +5~8 |

### 2.3 领主任务（Lord Quests）

在**城镇/城堡**中与领主交谈可接。领主任务涵盖军事和政治。

| 任务名 | 目标 | 奖励 | 关系影响 |
|---|---|---|---|
| **传递信件** | 将信件送达指定领主 | 30 第纳尔 | 任务发布者 +1，收件人 +1 |
| **收税** | 前往指定封地代为收税 | 税款的 1/5 | 领主 +3，封地 -2 |
| **追捕逃犯** | 前往村庄杀死藏匿的逃犯 | 金钱 + 荣誉选项 | 领主 +2，村庄 -1 |
| **训练部队** | 为领主训练一批士兵 | 金钱 | 领主 +4 |
| **借出军医** | 将同伴作为军医借给领主 | 金钱 | 领主 +5 |
| **借出同伴** | 将同伴借给领主完成任务 | 金钱 | 领主 +3 |
| **剿灭强盗巢穴** | 消灭领主领地附近的强盗据点 | 金钱 | 领主 +4 |
| **劫掠商队** | 攻击并劫掠指定势力的商队 | 战利品 | 领主 +10，目标势力关系 -5 |
| **跟踪间谍** | 跟踪在酒馆发现的间谍到秘密会面地点 | 金钱 | 领主 +5 |
| **揭发领主** | 公开指控一位领主的罪行 | 金钱 | 任务发布者 +7，被揭发者关系大幅下降 |
| **陷害领主** | 伪造证据陷害一位领主 | 金钱 | 领主 +关系，但荣誉 -2（可拒绝 +2 荣誉） |
| **俘虏敌方领主** | 在战场上俘虏指定敌方领主 | 金钱 | 领主 +关系 |
| **抓捕囚犯** | 捕获指定数量的特定势力战俘 | 金钱 | 领主 +关系 |
| **营救亲属** | 从敌方城堡/城镇中救出被囚的领主亲属 | 金钱 | 领主 +关系，女性贵族 +8 |

### 2.4 任务的结构性分析

| 任务类型 | 发布者 | 地点要求 | 核心奖励 |
|---|---|---|---|
| 乡村生存类 | 村长 | 村庄 | 村庄关系 |
| 商业民生类 | 行会长 | 城镇 | 城镇关系 + 金钱 |
| 军事类 | 领主 | 城镇/城堡 | 领主关系 + 金钱 |
| 政治类 | 领主 | 城镇/城堡 | 领主关系 + 荣誉选择 |
| 外交类 | 行会长/领主 | 城镇 | 多方关系 + 荣誉选择 |

**四变量交叉**：许多任务提供"荣誉选择"——接受报酬（+金钱）或拒绝报酬（+荣誉），直接体现了 Money-Honor 的张力。

## 三、宴会系统（Feasts）

### 3.1 概述

宴会是骑砍中最重要的社交场景之一——大量领主聚集在同一地点，是建立和维护关系的核心机会。

### 3.2 参加条件

| 条件 | 说明 |
|---|---|
| 友好/中立城镇举办的宴会 | 可自由进入 |
| 声望 > 200 | 无封臣身份也可参加 |
| 声望 < 200 | 需要贵族出身或被邀请 |
| 收到领主邀请 | 声望和关系足够高时会收到个人邀请 |

### 3.3 宴会期间可做的事

| 行动 | 效果 |
|---|---|
| 与领主交谈 | 维护和建立关系 |
| 政治站队 | 表达对候选人/政策的支持 |
| 求婚/联姻 | 向贵族女性示好 |
| 参加附带的比武 | 赢得声望和金钱，可将胜利献给女性 |
| 联盟谈判 | 非正式的外交协商 |

### 3.4 举办宴会

| 身份 | 举办限制 |
|---|---|
| 封臣 | 无冷却限制，可无限举办 |
| 国王 | 6 天冷却期（由配偶控制） |

举办成功的宴会可获得声望。

### 3.5 宴会的战略影响

- **社交集中**：大量领主暂时离开各自的巡逻/战斗位置聚集
- **军事真空**：参加宴会的领主不在前线，可能被敌方利用
- **持续数天**：如果主办者距离会场较远，宴会可能持续更长时间
- **信息交流**：是了解各方关系和立场的重要场合

### 3.6 对本项目的特殊价值

宴会是骑砍中"多 Agent 在同一时空社交互动"最集中的体现：
- 多个 Agent 同时在场，可以产生丰富的对话和互动事件
- 玩家可以观察 Agent 之间的自主社交行为
- 是产生"日记/思考链/社交网络"内容的天然场景
- 完美符合 AIvilization 架构中"Agent 内心可观察"的约束

## 四、联姻系统

### 4.1 求婚流程（战团）

```
拜访贵族女性（需要进入城镇/城堡）
    ↓
多次赠送诗歌、赢得比武并献给她
    ↓
关系值达到一定程度后表达求婚意愿
    ↓
获得监护人（通常是她的父亲/兄弟）批准
    ↓  批准条件：声望 ≥ 监护人声望的 1/2，关系友好
联姻成功
```

### 4.2 联姻的效果

| 效果 | 说明 |
|---|---|
| 家族联盟 | 与配偶家族建立政治联盟 |
| 宴会管理 | 配偶控制宴会举办的冷却时间 |
| 关系网扩展 | 通过配偶改善与其他领主的关系 |

### 4.3 霸主的联姻简化

霸主中联姻简化为：魅力检定 + 嫁妆谈判，流程更短但经济成本更高。

## 五、说服系统

### 5.1 说服的用途

| 场景 | 效果 |
|---|---|
| 劝说领主叛变 | 使敌方领主加入己方阵营 |
| 说服议和 | 促使交战势力停战 |
| 外交谈判 | 各种政治协商 |

### 5.2 影响说服成功率的因素

| 因素 | 影响 |
|---|---|
| Charm / Charisma | 基础说服能力 |
| Roguery 技能 | 增加不正当说服手段的成功率 |
| 性格对齐 | 你的行为风格与目标性格越匹配越容易说服 |
| 关系值 | 高关系降低说服难度 |
| 目标当前处境 | 处境越差（势力弱、关系差）越容易被说服 |

## 六、对极简化任务和社交设计的启示

### 6.1 任务系统的极简化方向

原版 55 种任务可归纳为以下核心类型：

| 极简任务类型 | 对应原版 | 核心功能 |
|---|---|---|
| **运送/护送** | 运送小麦/牛群/酒水/护送商队 | 经济行为 + 聚落关系 |
| **剿灭威胁** | 剿匪/处理掠夺者 | 军事行为 + 聚落关系 |
| **军事委托** | 训练部队/俘虏/营救 | 领主关系 + 声望 |
| **外交委托** | 传信/说服议和/揭发 | 领主关系 + 荣誉选择 |
| **经济委托** | 收税 | 金钱 + 关系代价 |

5 种极简任务类型覆盖了原版的核心功能。

### 6.2 宴会的极简化方向

| 保留 | 简化/移除 |
|---|---|
| 多 Agent 同一地点聚集 | 比武竞技（移除） |
| 社交互动产生关系变化 | 献花/诗歌等细节（移除） |
| 政治站队和外交协商 | |
| 军事真空效应 | |

宴会是 MVP 中最值得保留的社交场景。

### 6.3 社交系统的极简化核心

```
Agent 的社交行为 = 基于关系值做决策
    ↓
社交互动 → 关系变化 → 影响政治站队 → 影响战争走向
    ↓
每次互动都有实质后果（C3 交互有后果）
```
