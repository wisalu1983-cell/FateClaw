# 骑砍外交、政治与势力机制

> **文档用途**：详细记录骑砍中势力层级结构、外交决策流程、影响力系统和领地分封机制，作为 TinyMB 极简化势力与 NPC 自主行为设计（Phase 1 步骤 1.7）的输入材料。
>
> **数据来源**：Mount & Blade Wiki (mountandblade.fandom.com)、Bannerlord Wiki (fextralife)、Steam 社区讨论。战团和霸主并列对比。

---

## 一、势力层级结构

### 1.1 战团的势力结构

```
势力（Faction）
 └── 国王（Monarch）—— 势力领袖，拥有最终决策权
      └── 封臣（Vassals）—— 各自拥有领地和军队
           └── 聚落（Settlements）—— 归属于具体封臣
```

**六大势力**：
| 势力 | 国王 | 文化原型 |
|---|---|---|
| 斯瓦迪亚王国 | 哈劳斯国王 | 中世纪欧洲封建王国 |
| 罗多克王国 | 格拉维斯国王 | 瑞士/意大利山地城邦 |
| 维吉亚王国 | 亚罗格尔克国王 | 斯拉夫/俄罗斯 |
| 诺德王国 | 拉格纳尔国王 | 维京/北欧 |
| 库吉特汗国 | 桑嘉尔可汗 | 蒙古/游牧民族 |
| 萨兰德苏丹国 | 哈基姆苏丹 | 阿拉伯/中东 |

### 1.2 霸主的势力结构

```
王国（Kingdom）
 └── 国王（Ruler）
      └── 家族（Clan）—— 势力内的基本组织单位
           └── 领主（Lord）—— 家族成员中的活跃角色
                └── 聚落（Settlements）—— 归属于家族
```

**关键差异**：霸主引入了**家族（Clan）**层级，作为领主和势力之间的组织单位。家族拥有共同的领地和关系。

**八大势力**（帝国分裂为三部分）：

| 势力 | 特征 |
|---|---|
| 北帝国 | 元老院派 |
| 南帝国 | 正统继承派 |
| 西帝国 | 军事贵族派 |
| 弗兰迪亚 | 封建骑士王国 |
| 斯特吉亚 | 北方重步兵 |
| 阿塞莱 | 沙漠部落联盟 |
| 库赛特 | 游牧骑射 |
| 巴旦尼亚 | 森林游击 |

## 二、外交决策系统

### 2.1 战团的外交——国王主导

| 决策类型 | 决策者 | 机制 |
|---|---|---|
| 宣战 | 国王/元帅 | 国王或元帅直接决定，不经投票 |
| 议和 | 国王 | 国王可接受或拒绝和平提议 |
| 元帅任命 | 国王 | 国王直接任命 |
| 分封 | 国王 | 国王独自决定（但影响所有封臣关系） |
| 军事召集 | 元帅 | 元帅发出召集令，领主自行决定是否响应 |

**领主响应召集的因素**：
- 性格：正直型几乎必定响应，卑劣型可能忽视
- 与元帅的关系：高关系更愿意配合
- 当前状态：正在补给/恢复的领主可能延迟响应
- 距离：距离集结点远的领主到达更慢

### 2.2 霸主的外交——投票制

#### 影响力（Influence）系统

影响力是霸主中王国政治决策的"货币"。

**获取影响力**：

| 来源 | 收益 |
|---|---|
| 赢得战斗 | 按杀敌数计算 |
| 向驻军捐献士兵 | 按捐献量计算 |
| 领导大军参战 | 战斗期间获得 |
| 建造论坛（城镇建筑） | +1/天（视建筑等级） |

**消耗影响力**：

| 用途 | 消耗 |
|---|---|
| 召集大军 | 每位领主的召集成本取决于关系 |
| 维持大军凝聚力 | 防止大军自行解散 |
| 投票分封 | 票权与消耗的影响力挂钩 |
| 提议/投票政策 | 提出新政策或投票 |
| 宣战/议和 | 成功概率基于王国紧张度 |

#### 分封投票流程

```
攻陷聚落
    ↓
系统提名若干候选领主
    ↓
所有封臣投票（消耗影响力 = 增加票权）
    ↓
得票最高者获得聚落
```

#### 政策投票

| 政策示例 | 效果 |
|---|---|
| 土地税 | 国王获得 5% 村庄收入 |
| 国家垄断 | 国王获得 5% 聚落税收，但工坊产量 -10% |
| 圣王权威 | 国王每天 +3 影响力，非国王家族每天 -0.5 |

## 三、势力兴衰机制

### 3.1 势力强盛的正循环

```
军事胜利 → 攻占聚落 → 税收增加 → 养更多兵 → 更多军事胜利
                           ↓
                    领主满意 → 忠诚度高 → 不叛变
```

### 3.2 势力衰败的负循环

```
军事失败 → 聚落被占 → 税收减少 → 养不起兵 → 更多军事失败
                          ↓
                   领主不满 → 叛变 → 势力进一步削弱 → 灭亡
```

### 3.3 领主叛变——势力兴衰的关键转折点

领主叛变是势力格局变化的主要驱动因素。

**叛变的诱因**：

| 因素 | 权重 |
|---|---|
| 与国王的关系恶化 | 高——分封不公是主因 |
| 势力连续战败 | 高——"沉船鼠先逃" |
| 性格倾向 | 中高——卑劣/狡猾型更容易叛变 |
| 其他势力的拉拢 | 中——玩家或 AI 国王的外交努力 |
| 长期无封地 | 中——有功无赏导致不满 |

**叛变后果**：

| 项目 | 战团 | 霸主 |
|---|---|---|
| 领地 | 叛变领主带走自己的领地 | 叛变领主**不带走**领地 |
| 军队 | 带走自己的队伍 | 带走家族队伍 |
| 连锁效应 | 可能引发其他不满领主跟随叛变 | 类似 |

### 3.4 分封——政治系统最核心的矛盾

分封是骑砍政治系统中冲突最集中的点：

**战团的分封博弈**：
- 每攻占一块领地，国王需要决定分封给谁
- 获封者关系 +10
- **所有其他封臣** 关系 -2
- 这意味着每次分封都是"让一人开心、让所有人不满"
- 长期积累导致大量封臣对国王不满 → 叛变风险升高

**霸主的投票缓解**：
- 投票制分散了不满——不再全归咎于国王
- 但消耗影响力 = 间接消耗政治资本
- 频繁征战 + 频繁分封 = 影响力快速消耗 = 政治控制力下降

## 四、军事组织

### 4.1 大军（Army）系统——霸主独有

| 机制 | 说明 |
|---|---|
| 组建 | 消耗影响力召集多位领主 |
| 凝聚力 | 大军有凝聚力值，自然衰减，可消耗影响力补充 |
| 解散 | 凝聚力归零或领袖主动解散 |
| 移动 | 大军整体移动，速度由最慢成员决定 |

### 4.2 攻城流程

```
大军抵达敌方城镇/城堡
    ↓
建造攻城设备（攻城锤/攻城塔/投石机）—— 需要时间
    ↓
发动突击 或 等待饥饿投降
    ↓
攻城战（或自动结算）
    ↓
若胜利 → 进入分封流程
若失败 → 退兵，损失惨重
```

## 五、外交事件——Agent 行为的丰富来源

以下外交事件可以自然产生 Agent 的内心活动和社交互动：

| 事件类型 | Agent 可能的反应 | 产生的内容 |
|---|---|---|
| 分封决定 | 开心/愤怒/密谋叛变 | 日记、与其他领主的密谈 |
| 宣战 | 备战/犹豫/反对 | 军事部署思考、与盟友协商 |
| 议和 | 支持/反对/利用 | 政治站队、利益计算 |
| 领主叛变 | 震惊/同情/趁机拉拢 | 对叛变者的评价、自身处境反思 |
| 宴会邀请 | 赴宴/推辞/利用机会 | 社交规划、关系维护 |
| 攻城战 | 参战/避战/驰援 | 军事判断、忠诚度考验 |

这些事件是"Agent 内心可观察"的天然素材，也是驱动玩家兴趣的核心内容。

## 六、对极简化外交设计的启示

### 6.1 必须保留的核心机制

| 机制 | 理由 |
|---|---|
| 分封冲突 | 骑砍政治系统最核心的矛盾，产生丰富的 Agent 反应 |
| 领主叛变 | 势力格局变化的主驱动力 |
| 宣战/议和 | 势力间基本的外交行为 |
| 关系驱动的政治站队 | 关系值 → 行为的核心映射 |

### 6.2 战团 vs 霸主的选择

| 对比项 | 战团（国王决策） | 霸主（投票制） |
|---|---|---|
| 实现复杂度 | **低**——一个 Agent 决策 | 高——需要多 Agent 投票流程 |
| 戏剧性 | **高**——国王的每个决定直接产生不满 | 中——责任被分散 |
| 对 Agent 的要求 | 低——只需国王 Agent 做决策 | 高——所有封臣都需要投票决策能力 |
| 适合 MVP | **是** | 否（复杂度过高） |

**建议**：MVP 采用战团的国王决策模式，因为实现简单、戏剧性强、Agent 决策负担集中在国王身上。

### 6.3 极简外交骨架

```
势力 A ←—— 关系 ——→ 势力 B
  │                    │
  └── 封臣关系网 ──────┘
       │
  分封 → 不满积累 → 叛变（跳槽到更强势力/玩家阵营）
       │
  宣战/议和 → 基于实力对比和关系做决策
```
