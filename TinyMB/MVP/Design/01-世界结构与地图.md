# 世界结构与地图

> **所属**：极简骑砍规则集 · 模块 1
>
> **设计决策**：采用**单城镇方案**——整个游戏世界是一座自治城邦，骑砍原版的"大陆多城镇"体验转化为"城镇内多区域+多派系"体验。
>
> **决策依据**：`参考资料/单城镇方案核心机制验证.md` 确认骑砍 30 项核心机制中 29 项在单城镇下保留，唯一丧失项（城际套利）对核心体验无实质影响。

---

## 0. MVP 回标说明

本文档同时保留两层口径：

- **母稿**：完整单城镇 6 区结构，作为后续扩展参考。
- **首轮可验证原型回标稿**：明确首轮只启用 4 区、6 名命名 NPC 与轻量外部压力。

### 首轮可验证原型（MVP-Must）

- 活跃区域锁定为：**城主堡 / 市集区 / 酒馆区 / 外城农庄**
- 首轮命名 NPC 总数锁定为 **6 名**
- 外部威胁只保留**轻量替代物**：来访者、匪患传闻、商路异动、农庄求援、警戒状态变化

### 首轮原型后放（Later）

- 练兵场、城墙与城门作为独立区域
- 完整敌对城邦试探、围城、攻城与灾荒系统
- 区域控制权争夺与更长线的世界升级结构

## 1. 世界设定概述

**石冠城（Stonecrown）** —— 一座位于贸易要道上的自治城邦。

城邦内部三股势力长期博弈：军事贵族掌握城防和法律，商人行会控制经济命脉，平民公社掌握粮食和人力。三方各有优势、互相制约，任何一方都无法独自统治全城。城邦没有稳定的城主——城主之位本身就是三方争夺的焦点。

城邦外部存在持续不稳定因素：流窜匪患、外来者到访、敌对城邦的试探消息——这些外生压力会不断干扰城内平衡。首轮原型只实现其中的轻量信号与小事件，完整外患系统留待后续扩展。

玩家扮演的 Agent 以外来者身份抵达此城，从零开始在这个权力漩涡中找到自己的位置。

## 2. 城镇区域划分

石冠城母稿分为 **6 个功能区域**，对应骑砍原版的不同聚落功能；**首轮可验证原型只启用其中 4 区**。

```
              ┌─────────────────────────────┐
              │         城墙与城门           │ ← 防御/外部事件入口
              │      (Walls & Gates)        │
              └──────┬──────────┬───────────┘
                     │          │
          ┌──────────┴──┐  ┌───┴────────────┐
          │   城主堡     │  │    练兵场       │
          │ (Citadel)   │  │(Training Grounds)│
          └──────┬──────┘  └───┬────────────┘
                 │              │
          ┌──────┴──────┐  ┌───┴────────────┐
          │   市集区     │  │    酒馆区       │
          │  (Market)   │  │   (Tavern)     │
          └──────┬──────┘  └───┬────────────┘
                 │              │
              ┌──┴──────────────┴──┐
              │      外城农庄       │ ← 资源生产/城外
              │   (Farmlands)      │
              └────────────────────┘
```

| # | 区域 | 主导派系 | 核心功能 | 首轮回标 | 对应骑砍原版 |
|---|---|---|---|---|---|
| 1 | **城主堡** | 守备军团 | 政务厅（政治接触）、裁决入口、觐见/拜访权贵 | **MVP-Must** | 城镇城主大厅 + 城堡 |
| 2 | **市集区** | 商盟 | 交易市场、正规任务、雇佣服务、商路消息 | **MVP-Must** | 城镇市场 + 行会长 |
| 3 | **酒馆区** | 争夺区（中立） | 招募佣兵、打听消息、地下交易、社交聚会 | **MVP-Must** | 城镇酒馆 |
| 4 | **练兵场** | 守备军团（开放） | 士兵训练/升级、竞技比武、军事招募 | `Later` | 竞技场 + 军事训练 |
| 5 | **外城农庄** | 平民公社 | 粮食生产、农兵招募、基础物资采购、护民任务 | **MVP-Must** | 村庄功能 |
| 6 | **城墙与城门** | 共管（三方轮值） | 防御外敌、巡逻警戒、外部事件触发点 | `Later` | 大地图的"遭遇/防御"入口 |

首轮原型中的折叠处理：

- **练兵场 → Later**：基础兵源由外城农庄招募农兵与酒馆区招募佣兵承担
- **城墙与城门 → Later**：对外压力通过酒馆 / 市集的来访者事件、农庄求援和全局警戒状态替代

### 区域间移动

- Agent 在区域间移动需消耗**连续时钟中的时间**，但不再固定吞掉整整半天
- 首轮原型的移动耗时分为 **5 / 10 / 15 / 20 分钟** 四档，不同区域组合耗时不同
- 同一区域允许连续处理多件事务，因此玩家与 Agent 需要先规划“进这个区后顺手做什么”，避免来回浪费时间
- 移动途中有概率触发**随机事件**（遭遇其他角色、帮派冲突、偷盗、意外消息等）
- 有些区域存在**准入限制**：如城主堡需要一定声望或派系关系才能进入

| 路线 | 耗时 |
|---|---|
| 酒馆区 ↔ 市集区 | 5 分钟 |
| 城主堡 ↔ 市集区 | 10 分钟 |
| 城主堡 ↔ 酒馆区 | 10 分钟 |
| 市集区 ↔ 外城农庄 | 15 分钟 |
| 酒馆区 ↔ 外城农庄 | 15 分钟 |
| 城主堡 ↔ 外城农庄 | 20 分钟 |

## 3. 三大派系设计

3 个派系在城邦内形成**三角博弈**，对应骑砍原版的多势力结构。

### 守备军团（The Garrison）

| 维度 | 说明 |
|---|---|
| **身份** | 军事贵族及其武装力量，城邦的传统统治阶层 |
| **控制区域** | 城主堡、练兵场 |
| **核心资源** | 武力（城防军、精锐士兵）、法律权（逮捕/审判/分封） |
| **核心利益** | 维持秩序、扩张军事控制、确保城主之位属于军事贵族 |
| **弱点** | 经济依赖商盟税收，粮食依赖平民公社供给 |
| **领主性格倾向** | 正直型（忠诚守约）和好战型（崇尚武力）为主 |

### 商盟（The Merchant Guild）

| 维度 | 说明 |
|---|---|
| **身份** | 大商人、工坊主、贸易商组成的经济联盟 |
| **控制区域** | 市集区 |
| **核心资源** | 金钱（贸易利润、工坊产出）、情报（商路消息网络） |
| **核心利益** | 贸易自由、减税、市场规则由商盟制定 |
| **弱点** | 没有武装力量，安全依赖守备军团或雇佣兵 |
| **领主性格倾向** | 狡猾型（利益导向）为主，部分善良型（注重信誉） |

### 平民公社（The Commons）

| 维度 | 说明 |
|---|---|
| **身份** | 农民、工匠、退伍老兵、城区居民组成的自治联盟 |
| **控制区域** | 外城农庄 |
| **核心资源** | 人力（农兵兵源、劳动力）、粮食（城邦唯一的食物来源） |
| **核心利益** | 公平分配、反对压迫、保障平民权益 |
| **弱点** | 组织松散、装备落后、缺少政治话语权 |
| **领主性格倾向** | 善良正直型为主，少数暴躁/激进型 |

### 三角博弈动态

```
        守备军团
       /        \
  军事保护    武力压迫
  税收上缴    反抗起义
     /            \
  商盟 ———————— 平民公社
     经济剥削 / 劳动力供给
     物资采购 / 价格博弈
```

| 博弈场景 | 骑砍原版对应 |
|---|---|
| 守备军团+商盟 联手压制平民公社 → 粮食断供/民众暴动风险 | 贵族领主联合压榨村庄 → 村庄叛变 |
| 守备军团+平民公社 联手对抗商盟 → 经济停滞/资金链断裂 | 重视荣誉的领主打压奸商 → 贸易收入下降 |
| 商盟+平民公社 联手架空守备军团 → 城防空虚/外敌入侵风险 | 领主叛变导致势力虚弱 → 被其他王国攻占 |
| 三方暂时合作应对外部威胁 | 多势力联合抵抗入侵 |

## 4. 外部威胁机制

首轮可验证原型只保留**轻量外部压力**，用于证明世界不是静态舞台，但不直接实现完整攻城系统。

| 外部事件类型 | 频率 | 影响 | 首轮回标 | 骑砍对应 |
|---|---|---|---|---|
| **匪患传闻 / 农庄求援** | 常态 | 威胁农庄安全，驱动护民与巡逻任务 | **MVP-Must** | 野外遇到强盗 / 村庄被扰 |
| **外来者到访** | 随机 | 旅行商人、流浪佣兵、信使带来信息、机会和风险 | **MVP-Must** | 大地图上遇到商队 / 领主队伍 |
| **商路异动 / 警戒升级** | 1~3 天一次轻量信号 | 影响物价、任务和城内气氛，提醒玩家世界仍在变化 | **MVP-Must** | 战事传闻 / 边境紧张 |
| **敌方城邦试探** | 中频 | 外敌斥候出现→小规模冲突→可能升级为攻城 | `Later` | 敌国巡逻队 / 攻城前奏 |
| **攻城战 / 灾荒** | 低频高影响 | 围城、强制合作、大规模资源重分配 | `Later` | 攻城 / 战乱破坏 |

首轮原型中，外部压力的主要作用是：**给四区行动提供新的情报源、任务源和短期站队压力**。

完整的"内斗过度 → 外敌入侵 → 被迫合作"强制协作链路保留到 `Later`。

## 5. 玩家进阶路径

| 阶段 | 骑砍原版对应 | 城邦版表现 | 行为空间 |
|---|---|---|---|
| **外来者** | 自由冒险者 | 初到石冠城，无派系、无资源、无声望 | 酒馆找活干、接零散任务、打探消息 |
| **派系帮手** | 佣兵 | 为某个派系做事赚钱，不承担政治义务 | 以上 + 可进入该派系控制区域、接派系任务 |
| **派系核心** | 封臣 | 正式加入某派系，获得区域管理权，参与派系决策 | 以上 + 出席议事、管理下属、投票权 |
| **城主** | 国王 | 控制城邦的最高权力，需要平衡三方 | 所有行为 + 分配管理权、制定政策 |

首轮原型重点覆盖：**外来者 → 派系帮手**

`Later`：派系核心、城主

## 6. 参数设定

| 参数 | 值 | 理由 |
|---|---|---|
| 区域数 | 首轮 4 / 母稿 6 | 首轮压缩空间规模，但保留赚钱、社交/情报、派系接触、补给/平民四类必需场景 |
| 派系数 | 3 | 三角博弈最低门槛，支撑结盟/背叛/渔利等骑砍核心外交动态 |
| 首轮命名 NPC 数 | 6 | 保证三派系锚点 + 中立信息节点 + 最小灰度路线 |
| 母稿命名 NPC 数 | 12 | 供后续扩展为更完整的派系生态 |
| 区域间移动耗时 | 5~20 分钟/次 | 使“去哪、先去哪里、进区后顺手办什么”成为有意义的决策，同时避免强回合感 |
| 随机事件概率 | 移动时 20~30% 触发 | 让移动不只是"传送"，保留不确定性 |
| 完整外部攻城频率 | `Later` | 首轮不启用完整攻城循环 |

## 7. 裁剪决策记录

| 骑砍原版机制 | 极简化处理 | 判断 | 理由 |
|---|---|---|---|
| 多城镇大陆地图（22~57 城镇） | 单城镇，母稿 6 区 / 首轮启用 4 区 | **重构** | 核心机制验证确认 29/30 项机制保留，首轮进一步压缩到验证必需空间 |
| 三类聚落（城镇/城堡/村庄） | 合并为城镇内区域 | **合并** | 城堡无独占行动，村庄仅独占招兵 |
| 多势力（6~8 王国） | 3 派系 | **简化** | 3 方是三角博弈的最低门槛 |
| 开放大地图 + 自由移动 | 区域间移动（5~20 分钟/次） | **简化** | Agent 只需要“在哪个区域 + 正在移动/已到达”，同时保留空间选择压力 |
| 城际贸易套利 | **移除** | **移除** | 单城镇下无法实现，对 Agent 世界无设计价值 |
| 野外遭遇 | 移动随机事件 + 轻量外部压力 | **变形** | 从"荒野遇到"变为"城中遇到"、来访者、匪患传闻与农庄求援 |
| NPC 商队 | **移除** | **移除** | 无城际概念后不需要 |
| 多种地形/季节/天气 | **移除** | **移除** | 对 Agent 决策无价值 |
| 身份进阶终点（建立王国） | 改为"成为城主" | **变形** | 进阶结构保留，天花板从"建国"降为"掌城" |
