# 势力与 NPC 自主行为

> **所属**：极简骑砍规则集 · 模块 4
>
> **设计目标**：定义三大派系的内部结构、NPC 角色名册、NPC 自主行为模式、以及派系间的动态博弈规则。NPC 不是脚本驱动的背景板，而是与玩家使用同一套行为循环的独立 Agent。

## 3. NPC 自主行为模式

每个 NPC Agent 与玩家 Agent 使用**完全相同的行为循环**（见 `02-核心循环与行为空间.md`），区别仅在于：NPC 由 LLM 基于性格+目标+当前状态自主决策，有派系义务，且 NPC 之间会自发互动。

### 日常行为模板

| 行为类别 | 典型行为 | 触发条件 |
|---|---|---|
| **补给** | 去市集采购物资、去农庄补充粮食 | 粮食不足或随从需要补充 |
| **巡逻/执勤** | 去城墙巡逻、在本方区域维持秩序 | 外部威胁升高 或 日常轮值 |
| **社交** | 去酒馆交流、拜访其他NPC、举办宴会 | 关系需要维护 或 有政治议题 |
| **经济** | 经营产业、交易、收取费用 | 需要资金（商盟NPC常态） |
| **军事** | 训练随从、招募兵员、参加比武 | 军事需求或好战性格驱动 |
| **政治** | 参加议事、拉拢其他NPC、策划行动 | 派系势力值变化 或 重大事件 |
| **情报** | 收集消息、监视对手、打听动向 | 狡猾型NPC常态行为 |

### NPC 之间的自发互动

| 互动类型 | 触发条件 | 可能结果 |
|---|---|---|
| 对话/交流 | 两个 NPC 在同一区域 | 关系变化、情报交换、达成协议 |
| 争执/冲突 | 利益冲突 + 关系 < 0 | 关系恶化、可能升级为武力对抗 |
| 合谋/密谈 | 关系 > 30 + 共同利益 | 形成临时同盟、策划联合行动 |
| 比武/切磋 | 好战型NPC在练兵场相遇 | 声望变化、关系变化 |
| 交易/交换 | 双方有互补需求 | 资源转移、互惠协议 |

## 4. 派系内部机制

### 派系决策流程

1. 任一派系核心成员发起提案，消耗影响力
2. 该派系所有核心成员投票（支持/反对/弃权）
3. 多数赞成 → 通过；否则否决
4. NPC 投票倾向由性格+个人利益+与提案人的关系综合决定

### 派系内权力竞争

- 领袖不永久固定——决策持续让成员不满时可能面临弹劾
- 挑战方式：议事中发起"弹劾提案"或"领袖挑战"
- 玩家可以成为挑战者或被挑战者

### 叛变机制

触发条件（需同时满足多项）：
- 与当前派系领袖关系 < -30
- 与目标派系中某NPC关系 > 30
- 性格为狡猾型或卑劣型
- 目标派系势力值 > 当前派系势力值

后果：叛变者带走部分随从+情报 → 原派系势力值下降 → 新派系势力值上升。

## 5. 派系间博弈规则

### 区域争夺

6 个区域中，城主堡、市集区、外城农庄各有默认控制派系，酒馆区和练兵场为争夺区，城墙与城门三方共管。

争夺流程：议事决定"争夺 X 区域" → 派代表交涉 → 交涉失败可能升级为武力对抗 → 胜方获得控制权。

### 合作与联盟

- 两派系可达成临时联盟（持续 N 周），不争夺对方区域，外部威胁时优先协同
- 联盟可被单方面撕毁（代价：Honor↓，关系↓↓）

### 外部威胁的合作机制

外部威胁"危急"时：三方强制停战 → 贡献随从参与城防 → 胜利则声望↑↑+繁荣度↑；失败则所有派系势力值-10+繁荣度-20。

## 6. 事件驱动系统

| 事件类别 | 示例 | 频率 | 影响 |
|---|---|---|---|
| **外部军事** | 匪帮大规模袭扰、敌方城邦围城 | 每 20~30 天 | 迫使合作，消耗资源 |
| **经济波动** | 商路中断、物价暴涨/暴跌、工坊事故 | 每 7~14 天 | 经济压力，派系利益重分配 |
| **政治事件** | 城主选举、派系丑闻曝光、叛变传闻 | 事件驱动 | 权力格局变化 |
| **社交事件** | 宴会邀请、比武大赛、节日庆典 | 每 7~10 天 | 社交机会，关系重塑 |
| **个人事件** | NPC求助、同伴消息、旧友来访 | 随机 | 丰富叙事，创造选择 |

## 7. 裁剪决策记录

| 骑砍原版机制 | 极简化处理 | 判断 | 理由 |
|---|---|---|---|
| 6~8 个王国，多个家族 | 3 个城内派系，每派系 3 名 NPC | **简化** | NPC 总数控制在 12 |
| 领主独立队伍在大地图移动 | NPC 在城内区域间移动 | **变形** | 行为循环不变 |
| 家族（Clan）层级 | 移除 | **移除** | 单城镇内过度复杂 |
| 元帅召集 | 派系议事投票 | **变形** | 功能等价 |
| 婚姻/联姻 | MVP 移除 | **移除** | 不影响核心三角博弈验证 |
| 领主被俘/逃跑循环 | 简化版（关押+释放/赎回） | **简化** | 保留核心抉择 |
| 商队系统 | 移除 | **移除** | 外来商人以事件形式到访 |
