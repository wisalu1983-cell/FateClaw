# 圣杯战争的核心规则

> 本文档综合 Fate/stay night、Fate/Zero、Fate/hollow ataraxia、Fate/Apocrypha、Fate/Extra、Fate/Grand Order 等系列作品，提炼圣杯战争作为一套"仪式 × 对抗"系统的共性规则。特定作品的变体规则在文末单独标注。

---

## 一、圣杯战争的本质

圣杯战争（Holy Grail War）是一场围绕"圣杯"展开的大规模魔术仪式，其表层目标是争夺能实现愿望的万能之器——圣杯，而深层目的则是通过收集从者（Servant）的灵魂，打开通往"根源"（Akasha）的通道。

- **表层目标**：最后存活的一组 Master-Servant 获得圣杯，实现任意一个愿望。
- **深层目的**：利用被击败的从者灵魂回归"英灵之座"的力量，在世界上撕开通往"根源"的孔洞，并用大圣杯中积蓄的庞大魔力将该孔洞固定，实现通往世界外侧的通道。

从系统设计层面看，"圣杯能实现愿望"只是吸引魔术师参战的诱饵。Master 的核心功用是充当从者留存于现世的"依凭"（锚点），在从者召唤阶段之后，Master 对仪式而言就已完成使命。

## 二、参战构成

### 2.1 标准构成（冬木模式）

| 要素 | 数量 | 说明 |
|------|------|------|
| Master（御主） | 7 人 | 被大圣杯选中并获得令咒的魔术师 |
| Servant（从者） | 7 骑 | 从英灵之座召唤的英灵分身，分属七个职阶 |
| 监督者（Overseer） | 1 人 | 由圣堂教会派遣，负责规则监督和事件善后 |

### 2.2 职阶限制

每次圣杯战争的七个标准职阶互不重复：

- **三骑士**（上三位）：Saber、Lancer、Archer
- **四骑兵**（下四位）：Rider、Caster、Assassin、Berserker

原则上同一职阶只能召唤一骑从者。七骑从者全部被召唤后，其余候选 Master 失去参加资格。

## 三、圣杯战争的完整流程

### 第一阶段：魔力积蓄

大圣杯从所在地的灵脉中持续汲取魔力。由于过度抽取会导致灵脉枯竭，积蓄过程极其缓慢，通常需要约 **60 年**才能积累一次圣杯战争所需的魔力总量。

### 第二阶段：Master 选定

当贮存的魔力足以召唤七骑从者时，大圣杯自动开始选定 Master 候选人，并在其身体上刻下"预兆之痕"。选定遵循以下优先级：

1. **愿望强度**：内心（含潜意识）有强烈愿望要实现的人获得极高优先度。
2. **魔术资质**：拥有魔术回路是基础条件，但并非决定因素。
3. **缘分与因果**：与圣杯、与其他参与者之间的因果纽带会影响选定。

没有强烈愿望者即使是优秀魔术师，优先度也会很低。

### 第三阶段：从者召唤

Master 准备好触媒（圣遗物）后，举行魔术召唤仪式：

1. **大圣杯介入**：大圣杯连接"英灵之座"，搜索与触媒对应的英灵。
2. **职阶分配**：根据英灵的特性将其灌入预设的职阶框架，制作英灵分身。
3. **契约缔结**：英灵以从者形式现界，被强制接受"对令咒绝对服从"的约束，Master 获得三划令咒。

**触媒规则**：
- 有触媒：指定召唤与触媒有缘的英灵。若触媒对应多位英灵，则召唤与 Master 相性最好的那位。
- 无触媒：召唤与 Master 精神、性格、境遇相性最好的英灵。

### 第四阶段：战争开幕

七组 Master-Servant 齐聚后，圣杯战争正式开幕。

### 第五阶段：交战与淘汰

参战组之间互相交战，被击败或失去 Master 的从者其灵魂被小圣杯回收并扣留。战争持续到只剩最后一组为止。

### 第六阶段：圣杯降临

六骑从者的灵魂被注入小圣杯后，利用英灵灵魂"回归英灵之座"的力量在世界上穿孔，大圣杯的庞大魔力将此孔洞固定——这就是通往"根源"的门。在此过程中，胜者可使用圣杯实现愿望。

## 四、令咒系统

### 4.1 基本属性

| 属性 | 描述 |
|------|------|
| 来源 | 大圣杯赋予的魔力结晶 |
| 形态 | 三划构成的刻印，刻于 Master 肉体上 |
| 消耗 | 每使用一次消失一划，用完三次后令咒消失 |
| 形状 | 由 Master 魔术回路特性决定，一旦确定不可更改 |

### 4.2 令咒的两种用法

**用法一：绝对命令权**
- 对从者下达无法违抗的绝对命令。
- 命令越具体、越明确，效果越显著。（"立即移动到某地点"远强于"听我的话"）
- 从者在事后保留对命令的情感反应（例如厌恶御主）。
- 极少数意志极强的从者可对令咒做出有限度的抵抗。

**用法二：从者强化**
- 将令咒的魔力灌注于从者，实现瞬间的大幅能力强化。
- 可实现接近魔法的效果，如瞬间空间转移、宝具威力增幅等。
- 效果是暂时性的，但爆发力极大。

### 4.3 令咒的战略意义

- 令咒是 Master 控制从者的唯一强制手段。全部消耗后 Master 将失去对从者的绝对约束力。
- 令咒可以从他人手中夺取来补充画数。
- 令咒是 Master 参战资格的凭证；失去所有令咒通常意味着退出战争。

## 五、胜负规则

### 5.1 胜利条件

- 成为最后存活的一组 Master-Servant。
- 理论上胜者获得圣杯并可许愿。

### 5.2 淘汰条件

一组 Master-Servant 在以下情况下被淘汰：

| 情况 | 结果 |
|------|------|
| 从者被击败（灵核毁坏） | 从者消亡，灵魂被小圣杯回收；Master 失去参战伙伴 |
| Master 死亡 | 从者失去魔力供给源和现世依凭；除非找到新 Master，否则将逐渐消散 |
| Master 失去所有令咒 | 失去对从者的控制权和参战资格 |
| Master 主动弃权 | 可向监督者申请退出，接受教会庇护 |

### 5.3 允许的手段

圣杯战争中**没有禁止使用的手段**，只要遵守"神秘的隐匿"这一底线规则：

- 暴力、智略、卑劣的陷阱全部被允许。
- 可以结成同盟，也可以随时背叛。
- 可以针对敌方 Master（而非仅限从者对决）。
- 可以利用地形、结界、情报战等一切战术手段。

## 六、核心约束与禁忌

### 6.1 神秘的隐匿

这是圣杯战争中**唯一的绝对规则**。魔术世界最高准则是"神秘不可暴露于普通人"，战斗中若导致神秘泄露（如在公众面前展示魔术或超自然现象），监督者有权介入，严重者可能被魔术协会追查制裁。

### 6.2 监督者的职能

- **规则维护**：确保圣杯战争按规则进行，处理违规行为。
- **事件隐蔽**：协调善后工作，抹消战斗痕迹和普通人的记忆。
- **败退保护**：为弃权或落败的 Master 提供教会的庇护。
- **情报管理**：通过"灵器盘"掌握已召唤从者的数量和职阶信息。

### 6.3 从者真名的秘匿

从者的真名是最核心的情报。一旦真名暴露，对手可以：
- 推断出其宝具的种类和效果。
- 知晓其传说中的弱点。
- 制定有针对性的战术克制方案。

因此，隐藏己方从者真名、调查敌方从者真名，是圣杯战争中最基本的情报博弈。

## 七、圣杯的构成

### 7.1 大圣杯

| 属性 | 描述 |
|------|------|
| 本质 | 大规模魔法阵 |
| 功能 | 从灵脉汲取并储存魔力；制作从者分身；赋予 Master 令咒；最终阶段打开通往根源的通道 |
| 特性 | 圣杯战争的真正核心与动力源 |

### 7.2 小圣杯

| 属性 | 描述 |
|------|------|
| 本质 | 回收已死从者灵魂的容器 |
| 功能 | 保管从者灵魂，稳定大圣杯与现世之间的魔力通道 |
| 形态 | 可以是器物，也可以是被改造的人造人等载体 |

## 八、圣杯战争的变体规则

不同 Fate 作品中，圣杯战争在核心框架上保持一致，但在具体规则上存在差异：

### 8.1 大圣杯战争（Fate/Apocrypha）

- **阵营制**：14 骑从者分为红方（Red）和黑方（Black）两大阵营，每方 7 骑。
- **Ruler 职阶**：增设仲裁者（Ruler），不属于任何一方，负责监督和维持规则。
- **团队战**：阵营内部 Master 需要协作，阵营间展开大规模集团交战。

### 8.2 月之圣杯战争（Fate/Extra）

- **数字化**：在月球超级计算机"Moon Cell"内进行，参与者灵魂数字化。
- **淘汰赛制**：初始 999 人筛选至 128 人，采用一对一淘汰赛形式。
- **虚拟从者**：从者以数字实体的形式被召唤。

### 8.3 亚种圣杯战争

- 冬木以外的地域也曾出现过基于相似原理的小规模圣杯战争。
- 规模更小、规则更松散、圣杯的"纯度"和功能通常不及冬木正统圣杯。

### 8.4 迦勒底的从者召唤（Fate/Grand Order）

- 严格来说不是"圣杯战争"，而是人理修复机构通过类似技术召唤从者。
- 突破了"7 骑限制"，主人公可同时契约大量从者。
- 圣杯在各特异点以历史扭曲的触媒形式出现，而非传统的愿望器。

## 九、核心规则总结

| 规则维度 | 核心要点 |
|----------|---------|
| 参战资格 | 被大圣杯选中 + 成功召唤从者 + 获得令咒 |
| 参战人数 | 标准 7 组 Master-Servant |
| 胜利条件 | 最后存活的一组获得圣杯 |
| 控制机制 | 令咒三划——绝对命令 + 能力强化 |
| 信息博弈 | 隐藏己方真名，调查敌方真名 |
| 手段限制 | 无禁止手段，唯一约束为"神秘的隐匿" |
| 仪式本质 | 收集从者灵魂 → 打开通往根源的通道 |
| 时间周期 | 约 60 年一次（魔力积蓄所需） |
