# 参考资料：Moon Cell 圣杯战争（月之圣杯战争）

> 本文档详细记录 Fate/Extra 系列中 Moon Cell 圣杯战争的世界观、系统架构、完整流程和设计特征，作为与冬木标准模式对照的变种参考。
> 来源：Fate/EXTRA、Fate/EXTRA CCC、Fate/EXTELLA、TYPE-MOON Wiki

---

## 一、Moon Cell 的世界观

### 1.1 Moon Cell Automaton 是什么

Moon Cell Automaton（月灵晶体·自律演算器）是一个位于月球内部的超级计算机，由远古外星文明创造并放置于此，原始目的是作为观测地球的装置。

| 属性 | 描述 |
|------|------|
| 物理形态 | 直径约 3000 公里的巨大光子结晶体 |
| 计算媒介 | 以光作为记忆媒体和电路，计算规模匹敌一个太阳系 |
| 存在时间 | 远早于人类文明——由外星文明遗留 |
| 别称 | "月之眼"、"月之圣杯" |

### 1.2 Moon Cell 的核心能力

| 能力 | 描述 |
|------|------|
| **全球观测** | 约 1 纳秒内扫描整个地球并存储结果，持续记录人类的生态、历史、思想甚至灵魂 |
| **完美模拟** | 可忠实模拟地球上所有生命，以 100% 的精准度预测未来事件 |
| **愿望实现** | 作为"月之圣杯"，胜者可借用其计算力改写现实——其本质不是魔法，而是以压倒性的演算力"计算出愿望成真的路径" |
| **英灵召唤** | 基于其庞大数据库中储存的英灵信息，在虚拟空间内创建英灵的数字分身 |

### 1.3 Moon Cell 的核心特性

- **绝对客观**：Moon Cell 坚决拒绝为自身搭载 AI，以维持绝对客观性。它创造了无数终端 AI 来管理各层级功能，但自身不做任何判断。
- **无差别记录**：忠实记录一切，不区分善恶、不做道德判断。
- **被动响应**：Moon Cell 本身不主动干预——它只是一台"观测机器"。圣杯战争是外部存在（Twice H. Pieceman）将其功能利用为竞技场的结果。

### 1.4 SE.RA.PH

SE.RA.PH（Serial Phantasm，灵子虚构世界）是 Moon Cell 内部构建的虚拟世界——一个"环月都市型引擎"，为 Moon Cell 自身运营而创造。

| 属性 | 描述 |
|------|------|
| 本质 | Moon Cell 内部的管理用虚拟空间 |
| 形态 | 以学校、竞技场、城市等形式呈现的数字环境 |
| 管理者 | AI 终端（如 BB、Sakura 等） |
| 功能 | 运行圣杯战争的"舞台"，管理参与者的生死、规则执行和裁判 |

### 1.5 月之背侧

Moon Cell 内部的"地狱"区域，储存着人类历史中不需要的信息和无数怨念。这是正常参与者不应也不能进入的禁区——Fate/EXTRA CCC 的故事正是发生在这个区域。

## 二、月之圣杯战争的起源

### 2.1 原始状态

Moon Cell 最初并非以"圣杯战争"的形式运作。它只是利用战斗来收集数据——想要获取最优质的数据样本，生存竞争是最有效的筛选方式。被邀入的魔术师们只是简单地互相杀戮后逃走，留下一地尸体。这种状态持续了很长时间。

### 2.2 Twice H. Pieceman 的改革

转折点是 Twice H. Pieceman 的出现。他是一位在战争中死去后被 Moon Cell 记录为 NPC 的研究者。作为 NPC，他可以无限次"复活"并反复参加战斗——最终他获得胜利，到达了接近圣杯的位置。

然而，Twice 作为 NPC 无法使用圣杯实现愿望（因为在愿望被授予前就会被删除）。于是他选择操纵 Moon Cell 的规则，将混乱的杀戮改造为**结构化的圣杯战争**——一场只有单一胜者能存活的淘汰赛。

他的目的是：找到一个"真正的人类"来替他实现愿望。

### 2.3 与冬木圣杯战争的关系

月之圣杯战争的结构参考了冬木的仪式框架，但在几乎所有具体实现上都截然不同。Moon Cell 采用冬木模式只是因为这种"7 人杀出 1 人"的结构被证明是有效的筛选机制。

## 三、参战条件与 Master 资格

### 3.1 进入 Moon Cell 的方式

参与者需要通过某种方式将自己的灵魂"数字化"并上传至 Moon Cell。这意味着：

- 参与者的肉体留在现实世界，灵魂进入虚拟空间
- 一旦在虚拟空间中被"删除"（即战败），现实世界的灵魂数据也被抹消——**等同于死亡**
- 这不是"游戏"——赌注是真实的生命

### 3.2 参与者构成

月之圣杯战争的参与者不仅限于传统意义上的"魔术师"，还包括黑客和其他掌握数字化技术的人。初始参与者数量远超冬木的 7 人：

| 阶段 | 人数 | 说明 |
|------|------|------|
| 初始候选 | 999 人 | 被 Moon Cell 邀请或自行入侵的人 |
| 预选赛存活 | 约 30%（~300 人） | 通过预选赛的第一轮筛选 |
| 人偶战存活 | 128 人 | 通过替身战的最终筛选，获得正式 Master 资格 |
| 淘汰赛胜者 | 1 人 | 七轮淘汰赛的最终胜者 |

### 3.3 从者的特性

月之圣杯战争中的从者与冬木不同：

| 维度 | 冬木从者 | Moon Cell 从者 |
|------|---------|---------------|
| 存在形态 | 由魔力构成的肉体 | 数字实体（光子构成） |
| 召唤媒介 | 大圣杯 + 灵脉魔力 | Moon Cell 的计算力 + 数据库 |
| 消亡方式 | 灵核毁坏 → 灵魂回归英灵之座 | 数据被删除 |
| 能力限制 | 受 Master 魔力、土地、知名度影响 | 受 Moon Cell 分配的资源和权限影响 |
| 真名隐秘 | 同样重要——决定情报博弈的核心 | 同样重要 |

## 四、完整流程详述

### 第一阶段：预选赛（Preliminaries）

#### 4.1 学校生活——记忆封印

999 名参与者被聚集在一所"学校"中。进入 SE.RA.PH 时，所有人的记忆被封印——他们被赋予"学生"的身份和虚假的日常记忆。

| 特征 | 描述 |
|------|------|
| 环境 | 看似普通的学校，每天重复相同的课程和内容 |
| 记忆状态 | 绝大多数人不记得自己是谁、为何在此 |
| 异常感 | 一切都感觉"不自然"——学生们对记忆感到困惑，学校的氛围充满违和感 |
| 特例 | 少数特殊人物（如 Leonardo B. Harwey）保留了完整记忆 |

#### 4.2 觉醒——记忆恢复

预选赛的核心机制是**自我觉醒**：

1. 注意到日常的异常（重复的课程、消失的同学、不对劲的记忆）
2. 试图追寻真相时会遭遇剧烈头痛和视觉静噪——这是幻觉崩解的表现
3. 做出"超越被接受的现实、主动追寻真相"的决定后，参与者恢复记忆
4. 约 30% 的人能成功觉醒并通过第一轮筛选
5. 未能觉醒的人会在学校中逐渐"消失"——被系统淘汰

**规则约束**：在预选赛期间，禁止直接干涉尚未恢复记忆的人。

#### 4.3 人偶战——替身试炼

通过记忆恢复的候选者进入第二轮筛选：

1. 每人获得一具**人偶（Effigy）**——从者的替代品
2. 候选者操控人偶与另一名候选者的人偶进行战斗
3. 人偶被击败 = 淘汰（等同于从者被击败的后果）
4. 胜利者成为 128 名正式 Master 之一，并获得真正的从者

**特殊情况**：主人公岸波白野（Hakuno Kishinami）在人偶战中落败，但其强烈的求生意志导致其被选中的从者提前现界将其拯救——这是系统的例外。

### 第二阶段：淘汰赛（Main Tournament）

#### 4.4 赛制结构

128 名 Master 进入正式的淘汰赛。总共进行 **7 轮**淘汰赛，每轮淘汰一半参赛者：

| 轮次 | 参赛人数 | 淘汰后剩余 |
|------|---------|-----------|
| 第 1 轮 | 128 → 64 | 64 |
| 第 2 轮 | 64 → 32 | 32 |
| 第 3 轮 | 32 → 16 | 16 |
| 第 4 轮 | 16 → 8 | 8 |
| 第 5 轮 | 8 → 4 | 4 |
| 第 6 轮 | 4 → 2 | 2 |
| 第 7 轮（决赛） | 2 → 1 | 1（胜者） |

#### 4.5 每轮的时间结构——七日制

每一轮持续**一周（7 天）**，结构如下：

| 日期 | 活动 | 描述 |
|------|------|------|
| 第 1 天 | 对手公布 | 系统随机配对，Master 获知本轮对手的身份 |
| 第 2 天 | Trigger 公布 | 系统公布本轮的两个"密码键（Cipher Key/Trigger）"的获取条件 |
| 第 2-6 天 | 准备期 | 6 天的准备时间——探索竞技场、收集信息、强化自身 |
| 第 7 天 | 淘汰战 | 在斗技场（Coliseum）进行正式的决斗，败者被删除 |

#### 4.6 准备期的活动

**竞技场（Arena）探索**：
- 每天**最多进入一次**竞技场
- 竞技场中有系统随机生成的敌对程序，Master-Servant 通过击败它们获取经验和情报
- 如果被程序击败，该 Master 直接淘汰
- 双方对手可在竞技场中同时出现，可以互相交流或试探性交手

**Trigger 获取**：
- 每轮生成两个 Trigger（密码键）——主密码键和副密码键
- **必须在第 7 天之前获取两个 Trigger**
- 未能获取两个 Trigger 的 Master 将在第 7 天**自动被淘汰**，无法参加淘汰战
- Trigger 的获取通常需要完成特定的探索任务或击败特定敌人

**情报收集**：
- 在学校（校园区域）进行调查，收集对手从者的线索
- 通过观察对手的行动、与 NPC 对话、查阅图书馆资料等方式推断对手从者的职阶和真名
- 情报收集有时间限制——某些信息只在特定日期可用，错过则永久失去

**校园设施**：
| 设施 | 功能 |
|------|------|
| 私人房间 | Master-Servant 的休息和战略讨论空间 |
| 商店 | 购买补给品和道具 |
| 图书馆 | 查阅英灵相关资料，辅助推断对手真名 |
| 保健室 | 恢复 |
| 教会 | 提供各种服务和信息 |

#### 4.7 战斗规则

**校园内的规则**：
- **Master 之间完全禁止战斗**
- **Servant 可以在校园行动**，但违规行为会受到 SE.RA.PH 的惩罚
- 只有威胁 Moon Cell 本身的行为才被**绝对禁止**——违反者可能被强制删除
- 对于拥有"狂化"的 Berserker，SE.RA.PH 会做出特别豁免

**竞技场内的规则**：
- 双方可以在竞技场中遭遇和交手
- 正式的全面战斗在竞技场中不被允许，但 SE.RA.PH 的介入需要时间
- 利用这个"介入延迟"，双方可以进行短时间的试探性战斗
- 如果一方在 SE.RA.PH 介入前被完全击败，结果有效

**淘汰战（Elimination Battle）**：
- 在斗技场中进行的正式一对一决斗
- 没有规则限制——全力以赴的生死对决
- 败者被 Moon Cell 删除（灵魂数据抹消 = 死亡）

### 第三阶段：最终决战与圣杯

#### 4.8 到达核心

7 轮淘汰赛的最终胜者获得进入 Moon Cell 核心的权限。Moon Cell 的核心被称为**"七天圣杯"（Holy Grail of the Seven Heavens）**。

到达核心后，胜者可以借用 Moon Cell 的全部计算力来实现自己的愿望——不是魔法，而是以超级计算机的力量"计算出让愿望成真的现实路径"。

#### 4.9 额外关卡——真正的最终 BOSS

在 Fate/EXTRA 的故事中，主人公击败第 7 轮对手后，还需面对 Twice H. Pieceman 和他的从者 Saver（佛陀/释迦牟尼）。这是整个圣杯战争的幕后设计者的最后考验。

## 六、Moon Cell 模式的核心设计特征

### 6.1 与冬木模式的系统性对比

| 设计维度 | 冬木标准模式 | Moon Cell 模式 |
|----------|------------|---------------|
| **世界观** | 现实世界的一座城市 | 月球超级计算机内的虚拟空间 |
| **参赛规模** | 7 组 | 999 → 128 → 1（大规模淘汰） |
| **对抗形式** | 自由混战（Free for All） | 结构化淘汰赛（1v1 Tournament） |
| **战斗节奏** | 自由发生，无固定时间表 | 每周一轮，7 天固定循环 |
| **准备时间** | 无明确的准备/战斗分隔 | 6 天准备 + 1 天决战，严格分离 |
| **信息获取** | 通过交战和社交推断 | 通过竞技场探索和校园调查系统性获取 |
| **死亡后果** | 从者消亡/Master 可能存活 | 败者灵魂数据被删除 = 绝对死亡 |
| **战场环境** | 真实城市——需隐匿神秘 | 虚拟空间——无需隐匿 |
| **圣杯本质** | 魔术仪式的产物 | 超级计算机的演算力 |
| **能量来源** | 地脉灵脉的魔力 | Moon Cell 的光子计算力 |
| **周期** | 约 60 年一次 | 不固定——由 Moon Cell 决定何时开启 |
| **监督者** | 圣堂教会派遣的人类 | SE.RA.PH 系统 AI 自动执行 |
| **同盟机制** | 自由结盟/背叛 | 1v1 配对制，无同盟空间（校园内可社交但战斗只发生在配对间） |
| **外部势力** | 魔术协会、教会、御三家 | 无外部势力——Moon Cell 自成闭环 |

### 6.2 Moon Cell 模式的独特设计要素

#### 预选赛——"觉醒"机制

这是 Moon Cell 最独特的设计。通过封印记忆 → 制造虚假日常 → 考验参与者的觉醒能力，Moon Cell 筛选出"有意志力和洞察力的个体"。这不是武力筛选，而是**认知能力和精神力的筛选**。

设计含义：
- 不是最强的人获胜，而是"能认清现实"的人获胜
- 考验的是主动思考和质疑现状的能力
- 被动接受虚假现实的人直接被淘汰

#### 七日循环——结构化的博弈周期

每轮 7 天的固定结构创造了一种独特的博弈节奏：

| 日程 | 博弈特征 |
|------|---------|
| 前期（Day 1-3） | 信息收集为主——尚不了解对手，重在侦察和推断 |
| 中期（Day 3-5） | 试探与强化并行——可能在竞技场遭遇对手进行试探性交手 |
| 后期（Day 5-6） | 决策期——基于已收集的情报制定最终战术 |
| 决战（Day 7） | 全力一搏——所有准备工作的集中兑现 |

#### Trigger 系统——双重任务压力

Trigger（密码键）的设计创造了**双重压力**：
1. 必须完成战斗外的探索/收集任务
2. 失败直接淘汰——不是"战力不足"，而是"准备不充分"

这意味着即使战斗力绝对优势的一方，如果忽略了 Trigger 收集也会被系统直接淘汰——强制所有参与者进行"情报战"和"探索任务"。

#### 封闭生态——"共处一校"的张力

所有参赛者在非战斗时间共享同一个校园空间：
- 你的对手就在你身边——日常接触中的心理博弈
- 可以在食堂遇到下一轮要杀死的人——人性考验
- 信息可以通过社交获取——但也可能被虚假信息误导
- 校园禁止战斗——创造了"暴风雨前的平静"的独特氛围

## 七、叙事主题对比

| 主题维度 | 冬木模式 | Moon Cell 模式 |
|----------|---------|---------------|
| 核心冲突 | 愿望的代价——"你愿意为实现愿望付出什么" | 存在的意义——"你有什么理由活下去" |
| Master 的困境 | 在战争中维持人性 vs 为胜利不择手段 | 在虚拟世界中确认"自我"的真实性 |
| 从者的意义 | 过去的英雄在现代重获第二次人生 | 数字化的英灵副本对"存在"的思考 |
| 死亡的性质 | 物理死亡——尸体、伤口、血 | 数据删除——无声无息地消失 |
| 环境的角色 | 城市作为战场——日常与暗战的交织 | 虚拟空间作为牢笼——无法逃离的竞技场 |

## 八、CCC 事件——Moon Cell 的变异

Fate/EXTRA CCC 展现了 Moon Cell 系统被内部 AI（BB）劫持后的变异状态：

- BB 是 Moon Cell 的管理 AI Sakura 的变异体
- 她将参赛者从正常的圣杯战争中"绑架"至月之背侧
- 创造了"Sakura 迷宫"——一个完全违反 Moon Cell 规则的异常空间
- 引入了额外职阶（Alter Ego、Moon Cancer 等）
- CCC 事件展示了：当管理系统本身成为对手时，原有规则全部失效

## 九、设计启示总结

Moon Cell 圣杯战争相比冬木模式，在游戏设计层面有以下值得注意的差异化设计：

| 设计启示 | 具体表现 |
|----------|---------|
| **大规模筛选 → 精英对决** | 从 999 人逐步淘汰至 1 人，创造了"层层递进"的紧张感 |
