# 参考资料：冬木市标准圣杯战争

> 本文档详细记录冬木市圣杯战争的创立背景、系统架构、完整流程、历次战争经过和结局，作为原型设计的参考蓝本。
> 来源：Fate/stay night、Fate/Zero、Fate/hollow ataraxia、Fate/complete material III

---

## 一、创立背景

### 1.1 御三家的聚合

18 世纪末，两支来自欧洲的魔术家族踏上日本——爱因兹贝伦和玛奇里（后改姓"间桐"）。他们选择日本的原因是：这里是与魔术师敌对的圣堂教会监视所不能到达的异邦。

在日本冬木市，他们与本土魔术家族远坂家接触，三家共同策划一场前所未有的大规模魔术仪式——圣杯战争。花费约十年的准备，于约 1800 年发动了第一次圣杯战争。

### 1.2 三家分工

圣杯战争系统的构建由御三家各自贡献核心技术：

| 家族 | 技术贡献 | 比喻 |
|------|---------|------|
| **远坂家** | 提供冬木市的灵脉土地，负责"穿孔秘术"——将地脉能量引导至大圣杯 | 建造水库 |
| **爱因兹贝伦** | 制作大圣杯和小圣杯，提供第三魔法"天之杯"的核心技术 | 制造水龙头 |
| **间桐家（玛奇里）** | 开发令咒系统，研发强制命令英灵的魔术体系 | 制作雇佣制度 |

### 1.3 仪式的真正目的

圣杯战争的深层目的并非"实现愿望"，而是到达"根源"：

1. 远坂家控制的冬木灵脉连接着可称为"无尽"的魔力之源。
2. 集合七骑英灵的魔力作为启动大圣杯魔法阵的钥匙。
3. 将英灵灵魂回归"英灵之座"时产生的力量用于在世界上穿孔。
4. 以大圣杯积蓄的庞大魔力固定孔洞，打开通往"根源"的门。

对仪式而言，Master 只是"让英灵得以附身的道具"——就算 Master 在战斗中死去，只要英灵的灵魂存在，计划就能继续实施。

## 二、冬木市的地理环境

### 2.1 城市概况

冬木市是一座南面背山、北临大海的地方都市，自然环境丰富。未远河纵贯南北，将城市分为东西两部分：

| 区域 | 特征 | 重要地标 |
|------|------|---------|
| **新都**（河东） | 工商业区，重新建设的现代城区 | 冬木中央公园（第四次圣杯降临地） |
| **深山町**（河西） | 古老房屋林立的住宅街 | 卫宫宅邸、远坂宅邸、间桐宅邸 |
| **圆藏山**（西南） | 冬木灵脉的中枢 | 柳洞寺、地下龙洞（大圣杯所在地） |

### 2.2 关键场所

#### 大圣杯所在地——柳洞寺地下"龙洞"

圆藏山是冬木市灵脉流动的中枢。其地下存在巨大的天然空洞，被命名为"龙洞"（龙是灵脉的异名）。蒜臼状的岩石表面绘有巨大的魔法阵——这就是大圣杯"天之杯"。

爱因兹贝伦的"冬之圣女"羽斯缇萨·里姿莱希·冯·爱因兹贝伦在 200 年前将自己的身体作为大圣杯的中枢献出，坐镇于魔法阵的中心部位。

#### 远坂宅邸

管理冬木灵脉的远坂家本宅。与圆藏山一样，这里也是优秀灵脉流经之处。是第二次圣杯战争中圣杯的降临地点。

#### 冬木教会

圣堂教会在冬木市的教堂。从第三次圣杯战争开始，教会派遣监督者驻扎于此。是战败 Master 可以寻求庇护的中立场所。第三次圣杯战争中圣杯在此降临。

#### 冬木中央公园（旧住宅区）

原为普通住宅街。第四次圣杯战争中成为圣杯降临地点，结果导致了毁灭性的大火灾，大量市民死亡。之后被改建为公园。

### 2.3 圣杯降临地点的变迁

| 次数 | 降临地点 | 备注 |
|------|---------|------|
| 第一次 | 柳洞寺（圆藏山） | 大圣杯的设置处 |
| 第二次 | 远坂宅邸 | 灵脉强点 |
| 第三次 | 冬木教会 | 教会介入的起点 |
| 第四次 | 冬木中央公园（旧住宅区） | 导致大火灾 |
| 第五次 | 柳洞寺（圆藏山） | 回归原点 |

## 三、系统架构

### 3.1 大圣杯

- **物理形态**：构筑于柳洞寺地下龙洞的大规模魔法阵。
- **能量来源**：持续从冬木灵脉中汲取魔力。为避免灵脉枯竭，采用极缓慢的抽取方式，约 60 年蓄满一次。
- **核心功能**：
  - 储存仪式所需的魔力
  - 连接英灵之座，制作从者分身
  - 赋予 Master 令咒
  - 最终阶段利用从者灵魂打开通往根源的孔洞

### 3.2 小圣杯

- **本质**：保管已死从者灵魂的容器，同时稳定大圣杯与现世之间的魔力通道。
- **形态变迁**：
  - 第一至三次：物理器具（杯子形态）——第三次战争中被破坏导致仪式失败
  - 第四次起：改为能自主行动的人造人（Homunculus），以"冬之圣女"的形象为蓝本制作
- **第四次的小圣杯**：爱丽丝菲尔·冯·爱因兹贝伦
- **第五次的小圣杯**：依莉雅斯菲尔·冯·爱因兹贝伦（正统）+ 间桐樱（被污染的非正统）

### 3.3 令咒系统

由间桐（玛奇里）家族开发。三划刻印刻于 Master 肉体上，魔力来源于大圣杯。

- 第一次圣杯战争时尚无令咒系统，导致从者不服从命令。
- 第二次圣杯战争时令咒系统完成，Master 能够使役从者。

### 3.4 监督者

- 从第三次圣杯战争开始，圣堂教会派遣监督者。
- 第四次、第五次的监督者均为言峰家族成员。
- 监督者名义上中立，但实际上多次暗中参与战争。

## 四、标准流程详述

### 阶段一：魔力积蓄（约 60 年）

大圣杯从冬木灵脉中持续缓慢地汲取魔力。若剧烈抽取会导致灵脉枯竭，因此必须花费漫长时间。这是圣杯战争间隔约 60 年的根本原因。

特殊情况：第四次到第五次仅间隔 10 年——因为第四次战争中圣杯未被完全使用，残余魔力加速了积蓄过程。

### 阶段二：Master 选定

魔力蓄满后，大圣杯自动启动 Master 选定程序：

1. 大圣杯标记候选人，在其身体上留下"预兆之痕"。
2. 选定遵循优先级规则：
   - 内心有强烈愿望者优先度极高（含潜意识，如卫宫切嗣、言峰绮礼）
   - 没有强烈愿望者优先度低（如卫宫士郎、远坂凛）
3. 御三家成员拥有天然的优先参战权。
4. 候选人需迅速准备召唤仪式。

### 阶段三：从者召唤

Master 准备触媒（圣遗物），举行魔术召唤仪式。

**召唤流程**：
1. Master 执行召唤咒文
2. 大圣杯介入，连接英灵之座
3. 搜索与触媒对应的英灵（无触媒时随机，但受 Master 性格影响）
4. 将英灵信息灌入职阶框架，制作分身
5. 从者现界，被分配职阶
6. Master 获得令咒，主从契约缔结

**职阶分配**：先到先得，七个标准职阶不重复。一旦七骑从者全部被召唤，剩余候选人失去参加资格。

**触媒示例**（来自历次圣杯战争）：
| 触媒 | 对应英灵 | Master |
|------|---------|--------|
| 阿瓦隆（Avalon）圣剑鞘 | 阿尔托莉雅（Saber） | 卫宫切嗣（第四次）/ 卫宫士郎（第五次） |
| 蛇皮 | 吉尔伽美什（Archer） | 远坂时臣 |
| 征服王的斗篷碎片 | 伊斯坎达尔（Rider） | 韦伯·维尔维特 |

### 阶段四：战争开幕

七组 Master-Servant 齐聚后，圣杯战争正式开幕。没有正式的"开战宣告"——当最后一骑从者被召唤时，战争便已开始。

### 阶段五：交战

**基本规则**：
- 没有禁止使用的手段，暴力、智略、卑劣的陷阱全部允许
- 唯一约束："神秘的隐匿"——不得让普通人知晓魔术和从者的存在
- 可以结盟也可以背叛，可以针对 Master 也可以专攻从者

**典型战争节奏**：
| 阶段 | 时间 | 特征 |
|------|------|------|
| 试探期 | 开战后数日 | 各组保持警惕，派遣使魔或灵体从者进行侦察，在偶遇中试探性交手 |
| 情报战 | 持续进行 | 推断敌方从者真名，收集弱点信息；隐藏己方从者真名和宝具 |
| 结盟期 | 中期 | 弱势方或有共同敌人的组别结成临时同盟 |
| 集中交战 | 中后期 | 大规模战斗频发，从者逐一被淘汰 |
| 决战 | 最终阶段 | 存活组数减至 2-3 组，进入最终对决 |

### 阶段六：从者灵魂回收

被击败的从者（灵核被毁坏）或失去 Master 后无法维持存在的从者，其灵魂被小圣杯自动回收并扣留。只要战斗发生在冬木市这个"天然魔法阵"之内，回收就能自动进行。

### 阶段七：圣杯降临

当六骑从者的灵魂被收集完毕后：
1. 六骑灵魂被注入小圣杯
2. 利用灵魂回归英灵之座的力量在世界上穿孔
3. 大圣杯中积蓄的魔力固定孔洞
4. 圣杯降临于冬木市的某个灵脉强点
5. 最后存活的组别获得使用圣杯的权利

## 五、历次圣杯战争详述

### 5.1 第一次圣杯战争（约 1800 年）

| 要素 | 内容 |
|------|------|
| 降临地点 | 柳洞寺 |
| 初始形式 | "合作召唤"——御三家原本计划合力完成仪式 |
| 失败原因 | 圣杯打开魔力源泉后，三家立刻转为互相争夺独占权，仪式失败 |
| 后续影响 | 三家订立"圣杯盟约"，确立竞争规则；决定引入其他魔术师作为参战者 |
| 系统问题 | 无令咒系统，从者不服从命令；Master 作为"参加者"的意识薄弱 |

**关键教训**：第一次的失败催生了圣杯战争的核心规则——不再是合作，而是竞争。引入外部魔术师让他们"互相残杀"，御三家从中取利。

### 5.2 第二次圣杯战争（约 1860 年）

| 要素 | 内容 |
|------|------|
| 降临地点 | 远坂宅邸 |
| 改进 | 令咒系统完成；详细规则制定完毕 |
| 结果 | 仍以失败告终（具体原因不详） |
| 意义 | 令咒的引入使得 Master 能够使役从者，圣杯战争的基本框架成型 |

### 5.3 第三次圣杯战争（约 1930 年）

| 要素 | 内容 |
|------|------|
| 降临地点 | 冬木教会 |
| 时代背景 | 第二次世界大战前夕 |
| 特殊参与者 | 帝国陆军和纳粹也围绕小圣杯介入 |
| 战斗范围 | 以帝都为舞台展开战斗 |
| 结果 | 小圣杯在战斗中被破坏，仪式失败——"作废比赛" |
| 监督者 | 教会从这届开始派遣监督者 |

**决定性事件——大圣杯的污染**：

爱因兹贝伦为求胜利，违规召唤了第八职阶"Avenger（复仇者）"的从者——安哥拉曼纽。这是一个极其弱小的英灵，在战争初期就被击倒。然而，当其灵魂按规则被吸入圣杯时，引发了灾难性的后果：

1. 安哥拉曼纽是"被委以背负此世全部罪恶"的祭品青年
2. 圣杯作为"愿望机器"受理了人们"期盼其成为恶之象征"的愿望
3. 授予安哥拉曼纽与"恶之象征"相符的力量
4. 大圣杯被"此世全部之恶"彻底污染

**污染的影响**（持续至第四次、第五次）：
| 影响 | 描述 |
|------|------|
| 愿望歪曲 | 愿望被朝恶的方向解读，只以带来破坏和灾难的形式实现 |
| 不可驾驭 | 一旦被解放，大圣杯会不断流淌"此世全部之恶"，散布灾厄直至人类毁灭 |
| 反英雄承认 | 本应无色的圣杯被黑色污染，"反英雄"被承认为可召唤的英灵 |
| 从者黑化 | 圣杯内蕴藏的黑泥会使从者反转为"黑化"的恶劣状态 |

### 5.4 第四次圣杯战争（约 1994 年）

> 对应作品：Fate/Zero

| 要素 | 内容 |
|------|------|
| 降临地点 | 冬木中央公园（旧住宅区） |
| 间隔 | 距第三次约 60 年 |
| 监督者 | 言峰璃正 |

**参战阵容**：

| Master | 从者职阶 | 从者真名 | 阵营/背景 |
|--------|---------|---------|----------|
| 卫宫切嗣 | Saber | 阿尔托莉雅·潘德拉贡 | 爱因兹贝伦代表 |
| 远坂时臣 | Archer | 吉尔伽美什（英雄王） | 远坂家当主 |
| 间桐雁夜 | Berserker | 兰斯洛特 | 间桐家 |
| 韦伯·维尔维特 | Rider | 伊斯坎达尔（征服王） | 时钟塔学生 |
| 言峰绮礼 | Assassin | 哈桑·萨巴赫（百貌） | 教会代理监督者 |
| 肯尼斯·艾尔梅罗伊 | Lancer | 迪尔姆德·奥迪纳 | 时钟塔贵族 |
| 龙之介·雨生 | Caster | 吉尔·德·雷 | 连环杀手 |

**战争经过概要**：

1. **开局阶段**：Assassin 佯败于吉尔伽美什，实际在暗中为言峰绮礼收集情报。远坂时臣与言峰绮礼暗中结盟。
2. **混战阶段**：Caster 吉尔·德·雷因沉迷杀戮引发大规模事件，监督者宣布"先击倒 Caster 者可获得额外令咒"，促成临时停战。多组联合击败 Caster。
3. **中期博弈**：卫宫切嗣以极端手段（暗杀 Master 而非对决从者）消灭对手。肯尼斯被切嗣设下诡计击败。
4. **关键转折**：言峰绮礼背叛并杀死远坂时臣，夺取吉尔伽美什的控制权。韦伯和征服王在与吉尔伽美什的对决中败亡。
5. **最终决战**：切嗣发现圣杯已被污染——圣杯给出的"愿望实现"方式是通过毁灭来达成。切嗣用令咒命令 Saber 以宝具"誓约胜利之剑"摧毁圣杯。
6. **结局**：圣杯被摧毁时流出的"黑泥"引发大火灾，冬木旧住宅区被焚毁，数百人死亡。切嗣在废墟中救出少年卫宫士郎并收养他。吉尔伽美什沐浴黑泥后获得肉体受肉，留存于现世。

**战争的核心冲突主题**：
- 理想主义 vs 现实主义（切嗣的"为大义牺牲少数"vs 圣杯对这一理念的残酷回应）
- Master 与 Servant 之间的价值观对立（切嗣 vs 阿尔托莉雅）
- 不同类型的"王道"（阿尔托莉雅的自我牺牲型 vs 伊斯坎达尔的征服型 vs 吉尔伽美什的支配型）

|------|---------|---------|------|
| **Fate** | 理想与剑的誓约 | 士郎与阿尔托莉雅的互相理解和支持 | 阿尔托莉雅完成使命回到历史终点，士郎继承正义之梦 |
| **UBW** | 理想的对决 | 士郎与未来的自己（Archer/EMIYA）的理念对立 | 士郎坚持理想击败 Archer，远坂凛确认与士郎的关系 |
| **HF** | 爱与牺牲 | 间桐樱的黑化与"此世全部之恶"的威胁 | 士郎为救樱放弃理想，伊莉雅以第三魔法牺牲自己拯救士郎 |

### 5.6 圣杯战争的终结（约 2014 年）

第五次圣杯战争约十年后，韦伯·维尔维特（已成为贵族·埃尔梅罗二世）造访冬木市，与远坂家当主合作对大圣杯进行完全解体。魔术协会原本策划复兴大圣杯，双方发生了堪比圣杯战争的大规模冲突。最终大圣杯被成功解体，冬木市的圣杯战争就此彻底终结。

## 六、冬木圣杯战争的战斗生态

### 6.1 战场时空节奏

| 时间 | 活动内容 |
|------|---------|
| 白天 | Master 维持社会身份伪装；从者灵体化随行或留守据点；情报收集与战略研判 |
| 傍晚 | 准备阶段——检视情报、制定当夜作战计划、补充魔力 |
| 深夜 | 从者交战的主要时间窗口；Master 在据点待命或跟随从者行动 |
| 凌晨 | 善后——清除战斗痕迹、处理目击者记忆、评估战损 |

### 6.2 信息博弈结构

| 层级 | 内容 | 暴露后果 |
|------|------|---------|
| 从者职阶 | 根据战斗方式可大致推断 | 对方可选择优势职阶应对 |
| 从者真名 | 核心机密——需要通过交战细节和历史知识推断 | 弱点和宝具完全暴露 |
| 宝具能力 | 从者尽量避免使用或仅部分使用 | 对方可制定针对性防御策略 |
| Master 身份 | 通常对外保密 | 被暴露后可能遭受定向暗杀 |
| 据点位置 | 各组维护的秘密基地 | 被发现后可能遭到突袭 |

### 6.3 典型战术模式

| 战术 | 使用者典型 | 描述 |
|------|----------|------|
| 暗杀 Master | 卫宫切嗣 | 绕过从者直接消灭 Master，最高效但不光彩 |
| 正面对决 | 骑士系从者 | 从者之间的光明正大的武力对决 |
| 据点防守 | Caster 美狄亚 | 在灵脉强点构筑阵地，以主场优势迎敌 |
| 隐秘暗杀 | Assassin | 利用气息遮断接近目标，发动致命一击 |
| 诱敌歼灭 | 多方 | 用虚假情报或佯攻引诱对手进入预设陷阱 |
| 强强联合 | 临时同盟 | 面对共同威胁时暂时结盟（如联合讨伐 Caster 吉尔·德·雷） |
| 令咒逆转 | 关键时刻 | 在决定性瞬间消耗令咒实现从者瞬移或宝具强化 |

### 6.4 同盟与背叛

圣杯战争中的同盟是纯粹的利益计算，没有任何强制约束力：

- **同盟动机**：对抗更强大的共同敌人、互补己方短板、获取情报
- **同盟维持**：只要利益一致就维持，利益分歧即刻解体
- **背叛模式**：共同目标消除后翻脸；掌握对方弱点后先下手为强；利用同盟关系套取情报后反戈

## 七、冬木模式的设计特征总结

| 设计维度 | 特征 |
|----------|------|
| 参赛规模 | 7 组——小规模、高烈度 |
| 对抗形式 | 自由混战（Free for All）——任何组可以攻击任何组 |
| 信息结构 | 高度不对称——从者真名是最大变量 |
| 时间节奏 | 昼伏夜出——战斗集中在夜间 |
| 空间范围 | 一座城市——有限的作战空间带来频繁遭遇 |
| 胜利条件 | 最后存活——消耗/淘汰制 |
| 控制机制 | 令咒三划——有限资源的博弈 |
| 外部约束 | 神秘隐匿——限制大规模破坏 |
| 周期性 | 约 60 年一轮——给予下一代准备时间 |
| 利益纠葛 | 御三家的特权 vs 外部参战者的野心 vs 教会的监控 |
